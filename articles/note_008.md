# 【AI対話アプリ】200の問い #008「モチベーション源泉分析」

このアプリは「モチベーション源泉分析」のためのAI対話プロンプト集です。モチベーションは「気分」ではなく「源泉」を知ることで、継続的に湧き上がる。その源泉が何かを言語化する。

---


## 1. 「達成感」と「貢献感」と「自由感」で源泉を分類する

モチベーションの源泉は人によって全く異なります。AIに「あなたが『よし、やるぞ』と思う瞬間はどんな時ですか」と聞きます。その答えの中に「達成感」という言葉が出てきたら「成功を求める人」。「貢献感」なら「他者を助けることで動く人」。「自由」なら「自主性を求める人」。「認言」なら「評価を求める人」。私の場合「困っている部下を助けられた時」という答えが出て、自分の源泉は「貢献感」だったことに気づきました。その気づきで仕事の選び方が劇的に変わり、同じ職場でも関わり方が変わりました。


## 2. 「モチベーション低下」の原因は源泉を活かしていないこと

モチベーションが下がるのは「やる気がない」からではなく「自分の源泉に合った環境にいない」からです。AIに「最近モチベーションが低い理由を『今の環境が自分の源泉を満たしていないから』という視点で分析してください」と頼みます。例えば「貢献感が源泉」なのに「個人ノルマのみ」の営業をしていれば当然低下します。私は「貢献感が源泉」なのに「経営層との政治的な会議ばかり」の日々が続いていて、それが低下の原因だったことに気づきました。その気づきで「月1回は部下育成の時間を『義務』として組み込む」という行動が生まれました。


## 3. モチベーションは「循環」させることで持続する

モチベーションは「頑張り続ける」ものではなく「循環」させるものです。高い時期低い時期を理解することで低い時期も対応できます。AIに「あなたのモチベーション曲線（過去1年間）を思い出してください。どの月が高くてどの月が低かったですか、そしてなぜですか」と聞きます。そのパターンを認識することで「来月が低くなるのは予測可能」と分かり事前に対応できます。私は「3月は人事異動で低い」「7月は夏季休暇で低い」という循環を認識し事前に「その時期は部下との面談を増やす」という工夫をするようになりました。


## 4. 外部からのモチベーションと内部からのモチベーション

モチベーションには「外部要因（給料評価他人の認言）」と「内部要因（自分の価値観成長実感）」があります。AIに「あなたのモチベーションの何%が外部要因で何%が内部要因ですか」と聞きます。その比率が見える化されると「もし給料が減ったら」「もし異動したら」という変化への対応力が高まります。私は「自分のモチベーション30%は給料。でも70%は『成長実感と部下育成』」という比率に気づき、その70%を最大化する仕事の選び方ができるようになりました。


## 5. 源泉に合わせた「環境設計」をする

見えた源泉に合わせて環境を設計することで、モチベーションは継続します。AIに「あなたのモチベーション源泉『◯◯』を最大限発揮できる環境は、どんな環境ですか。その環境を今の職場で『部分的に』作るには何をすべきですか」と聞きます。完全に職場を変えなくても「部分的に源泉が満たされる環境」を作ることで、モチベーション低下を防げるのです。


## 体験談: 「やる気が出ない」から「自分のモチベーション源泉を知って継続的に動く」へ

最初このアプリを使った時、自分のモチベーション源泉は「給料」だと思っていました。でも5回の深掘り質問で「実は『困っている人を助ける瞬間』が最もモチベーション高い」ことに気づきました。その瞬間から「あ、自分は『給料のために働く人』ではなく『貢献のために働く人』なんだ」と自己理解が変わった。その理解の後、毎月の仕事の選び方が変わりました。数字にならない「部下育成」や「組織課題の解決」に時間を使うようになり、それでも給料は上がりました。なぜなら、その「貢献活動」が組織に大きな価値をもたらしていたから。モチベーション源泉を知ることで給料と貢献の両立ができるようになったのです。

## 🤔 5つの問い (このアプリの中核)

1. 「よし、やるぞ」と思う瞬間はどんな時ですか
2. 最近モチベーションが低かった理由を、源泉が満たされていない視点で分析してください
3. あなたのモチベーション源泉は「達成」「貢献」「自由」「認言」のどれが強いですか
4. その源泉を今の環境で最大限活かすには何をすべきですか
5. モチベーションが低い時期を「無理に上げる」のではなく「循環」として受け入れるなら今月はどう過ごすべきですか
