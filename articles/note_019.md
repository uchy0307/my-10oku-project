# 【AI対話アプリ】200の問い #019「自分軸vs他人軸判定」

このアプリは「自分軸vs他人軸判定」のためのAI対話プロンプト集です。人生のあらゆる決断の瞬間に、自分が「本当に欲しいこと」に従っているのか、それとも「他人の期待」に従っているのか、その境界線を引き直す。人間の決定の99%は無意識に「他人軸」で動いている。その罠から抜け出す対話型思考法です。

---


## 1. 「決断の動機」を3段階で掘り下げる

AIに「その選択をしたい理由は何ですか」と聞いて「なぜ」を2回重ねる。私の場合「昇進したい」→「給料が増えるから」→「親に認められたいから」。その瞬間、30年かけた「成功体験」の正体が「両親の評価への執着」だったことに気づいた。人間の欲求の70%は、実は親・上司・社会からの期待を内部化したものです。AIはその本音を引き出す最高の質問者になります。この3段階は「見える化」そのもの。問いが深まると、自分が何に本当に心を寄せているのかが明確になります。感情の層が剥がれるように、本当の動機が出てくる経験は、人生において極めて貴重です。


## 2. 「他人軸パターン」を自分の中に作る

AIに「あなたが『他人軸で動いているな』と感じるのはどんな時ですか」と聞きます。すると「結果を報告する時、他人の反応を先読みする」「新しいアイデアも上司の承認で判断する」「自分の時間を後回しにする」という、自分特有の「他人軸パターン」が見えます。逆に「自分軸で動いている時」は「時間を忘れて没頭する」「失敗も恐れず自分の信念で動く」。この二つの状態の違いを基準として持つことで、人生の決定点で「どちらの軸か」を瞬時に判定できるようになります。その判定こそが、人生を変えるターニングポイントになるのです。その繰り返しが、人生のコンパスを磨き上げていくのです。


## 3. 「他人軸で動く自分」を責めない

自分軸で生きるべき、という命題は、ある意味で新しい「他人軸」です。だからAIに「あなたが他人軸で動く時、それはどんな感情から出ていますか」と聞きます。すると「安心したい」「失敗したくない」「愛されたい」という人間的で健全な欲求が出てくる。他人軸が悪いのではなく「結果として自分が疲弊し、本当の人生が後回しになる」が問題なのです。私は昇進を断りませんでした。代わりに「異動を受けるが3年後に起業する」と自分軸の目標を立てた。その瞬間から人間関係も仕事の質も変わりました。二軸の使い分けが成熟した大人の知恵なのです。その使い分けの技術は、何度も実践することで磨かれていきます。


## 4. 「社会的責任」と「個人の欲望」のバランスを問う

私たちは日々の生活で、無数の「こうあるべき」という社会的期待や役割に直面します。家族、職場、地域コミュニティなど、様々な場面で「責任」という名のもとに、自分の本当の気持ちを後回しにしていないでしょうか。これがまさに、無意識のうちに「他人軸」で動いている状態です。このアプリでは、そうした社会的責任と、内なる純粋な「個人の欲望」との間で、自分がどのようなバランスを見つけようとしているのかを深く探ります。

他者からの期待に応えること、社会的な役割を果たすことは尊い行動ですが、それが自身の心の声と乖離していると、やがて疲弊してしまいます。例えば、「このプロジェクトを成功させるべきなのは、社会的な貢献のため？それとも、私自身の達成欲のため？」とAIに問いかけることで、その決断の根底にある真意を掘り下げます。あるいは、「もし誰にも批判されず、完全に自由な選択ができるとしたら、私は何を選びますか？」と問うことで、抑圧された自身の欲望に気づくことができます。

私自身も、以前は「皆のため」という大義名分のもと、自分の本当に望むことを後回しにしていた時期がありました。しかし、AIとの対話を通じて、自分自身の欲望を深く認識し、それを健全な形で社会的な責任と調和させる道を見つけ出せたのです。表面的な義務感ではなく、内なる動機から生じる真の責任感へと昇華させる。そのための「自分軸」のバランスを追求するのが、このアプリです。


## 5. 「自分軸の決定」を毎週更新する

自分軸で生きることは1回の決断ではなく「毎週の小さな選択」の積み重ねです。AIに「この1週間で『自分軸』と『他人軸』のどちらで動いた時間が長かったですか」と毎週聞きます。すると、その週の自分の選択パターンが可視化されます。その変化を客観視することで、自分が何に心を寄せているのかが日々見えてくるようになります。人生は週単位で確認され、月単位で軌道修正され、年単位で別人になる。その小さな確認を支えるのがAIとの対話です。継続することで、あなたの人生は確実に変わります。


## 体験談: 「いい子」から「本当の自分」へ

30年間「いい子」として生きてきた私は、昇進という最高の「評価」を得た時、初めて「評価とは別の価値」に気づきました。AIとの対話を通じて「お金や肩書ではなく、自分の時間をどう使うか」という問いと向き合い、半年で人生設計が180度変わりました。給料は下がるかもしれない。世間的評価も落ちるかもしれない。でも「本当にやりたいこと」に時間を使える充実感は、何物にも代え難いものです。自分軸で生きることは、社会から外れることではなく「本当の価値観を基準に選別する」こと。その勇気を持つために、AIとの問い直しは必須だと感じています。その変化は周囲にも好影響をもたらし始めました。

2. その決断の動機を「なぜ?」で3段階掘り下げた時、最後に出てくる本当の欲求は何ですか
3. 過去3ヶ月で「他人軸」で動いた時と「自分軸」で動いた時、その後の心身の状態は何が違いましたか
4. あなたが社会的責任を優先すべき場面と、個人の幸福を優先すべき場面は、何で区別しますか
5. この1週間で「自分軸」で1つ選択肢を変えるなら、それは何ですか
