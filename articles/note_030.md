# 【AI対話アプリ】200の問い #030「リーダーシップ診断」

---


## 1. 「あなたのリーダーシップタイプ」を診断する

AIに「あなたが部下に指示をする時『どんな言葉かけをしますか』『部下とどんな関係を作っていますか』『決定を下す時『何を最優先していますか』」と聞きます。その問いから「指示型リーダー」「育成型リーダー」「民主的リーダー」など「あなた特有のリーダーシップタイプ」が見えてくるのです。その診断が「あなたのリーダーシップの土台」になります。


## 2. 「部下から見たあなた」を問う

AIに「あなたの『親友』『部下』『上司』『同期』が『あなたのリーダーシップについて何と言うと思いますか』を推測してください」と聞きます。その推測から「自分が思っているリーダーシップ」と「他者から見たリーダーシップ」のギャップが見えるのです。その「見えないギャップ」こそが、リーダーシップを限定している最大の要因なのです。


## 3. 「リーダーシップの弱み」を認識して対策を打つ

AIに「あなたのリーダーシップの『最大の弱み』は何だと思いますか。その弱みが『チームの成果』にどう悪影響を与えていると思いますか」と聞きます。すると「自分は『意思決定が遅い』という弱みを持つが『これが営業チームの機動力を削いでいた』」というような「弱みの悪影響」が見えてくるのです。その見える化が「改善」のスタートラインになります。


## 4. 「理想のリーダー像」を描いて逆算する

AIに「あなたが『理想とするリーダー』は誰ですか。その人の『何がいいと思いますか』『あなたとの違いは何ですか』」と聞きます。その問いから「自分は『人の話を最後まで聞く習慣』がないが『理想のリーダーは『その習慣』を持っている」というような「改善の具体像」が見えてくるのです。その具体像から「毎日1つのアクション」へと落とし込むことで、リーダーシップは確実に育つのです。


## 5. 「チーム成果との相関」を振り返る

AIに「この半年で『あなたのリーダーシップが変わった』と感じることはありますか。そしてそれが『チームの成果』にどう影響していますか」と聞きます。その振り返りを毎月繰り返すことで「あなたのリーダーシップの成長」と「チームの成長」の相関が見えてくるのです。その相関の見える化こそが「真のリーダーシップ」の実感になるのです。自分の成長がチーム全体の成長につながる喜びは、人生で最高の充実感なのです。


## 体験談: 診断で「本当のリーダーシップ」が開花

「部下のモチベーションが低い」という悩みを持っていました。でもAIとの診断で「自分は『部下の意見を最後まで聞かない癖がある』『すぐに自分の正解を押しつけている』」という弱みに気づきました。それを改善したら、部下から「自分たちの意見が聞かれている」という実感が生まれ、チームの成績が3ヶ月で50%上がったのです。リーダーシップは「テクニック」ではなく「自己認識」から始まるのです。自分を知ることから、人を導く力が生まれるのです。

リーダーシップ診断の最終ステップは『あなたが理想とする5年後のリーダー像』をAIに描かせることです。今の自分との差分が、次の半年の学習テーマになる。私は『静かな指揮官』を理想にしました。怒鳴らず、急がず、しかし芯がぶれない。その一行を毎週見返すだけで、判断と振る舞いが整うのを実感しています。


## ✅ 5項目セルフチェック

- [ ] 自分のリーダーシップタイプを「言語化」できた
- [ ] 「親友・部下・上司」から見たあなたのリーダーシップを推測した
- [ ] 自分のリーダーシップの「最大の弱み」を認識した
- [ ] 理想とするリーダーと「自分の違い」を具体化した
- [ ] リーダーシップの成長と「チーム成果」の相関を振り返った


## 🤔 5つの問い (このアプリの中核)

1. あなたのリーダーシップタイプは「指示型」「育成型」「民主的」どれですか
2. 部下から見たあなたのリーダーシップは「何が強みで、何が弱みだと思いますか」
3. あなたのリーダーシップの「最大の弱み」が「チームの成果」にどう悪影響していますか
4. あなたが理想とするリーダーの「何をいいと思いますか」「自分との違いは何ですか」
5. この半年で「あなたのリーダーシップ」の成長と「チーム成果」の相関は何ですか

## 📖 追加体験談・失敗回避リスト

- ❌ AIの回答をそのまま貼り付けて終わらせない（必ず一晩寝かせて自分の言葉に翻訳）
- ❌ 1回の対話で結論を急がない（最低3回、角度を変えて問い直す）
- ❌ 完璧な回答を待たない（70%の手応えで動き、残り30%は実践で埋める）
- ❌ 一人で抱え込まない（週1回、家族か仲間に1分だけ進捗を語る）
- ❌ 30日で変化が見えなくても焦らない（深い変化は90日後に表面化する）
