# 【AI対話アプリ】200の問い #052「友人の整理・選別」

このアプリは「友人の整理・選別」のためのAI対話プロンプト集です。SNS時代に「つながり」は増えたのに、心が軽くなるつながりは減ったあなたへ。誰と時間を使うかを言語化し、関係の質を劇的に改善するための問いです。

金曜夜の居酒屋。47歳の僕は、8人の同級生に囲まれながら、心の中で数えていた。「この中で、本当に話したい人は何人だろう」。飲み会は楽しい。盛り上がっている。でも帰路のタクシーの中で、いつも虚脱感が残る。SNSではフォロワーが500人を超えたのに、22時に「今夜誰かと話したい」と思ったとき、電話できる相手は3人しかいない。その3人と8人の間にある「何か」を整理したくて、スマホを開いた。

## 1. 「充電関係」と「消費関係」の二分法で見える化する

友人関係の質を判定する最初のステップは、シンプルな二分法です。あなたが相手と時間を過ごした後、心が満たされるか、それとも疲れるか。この「充電」と「消費」の感覚は、極めて正直です。AIに「私の現在の友人関係10人について、以下の3点で分析してください: ①会った後、心が軽くなるか重くなるか ②相手は私の話をどれだけ聴いているか ③この関係から1年後も深まる可能性」と依頼します。驚くほど冷徹な分析が返ってきます。僕の場合、同級生8人のうち「充電関係」は2人。残りの6人は「消費関係」でした。その瞬間、金曜夜の虚脱感の正体が分かった。意識的な関係選別の第一歩です。

## 2. 「呼んでくれる人」を最優先に再編成する

関係性には目に見えない上下関係があります。呼ぶ側と呼ばれる側。大事な判断基準の一つは「相手が私を呼ぶかどうか」です。AIに「私が相手を誘うばかりで、相手から誘われない関係は、心理的にはどのような状態ですか」と問うと、冷たい答えが返ってきます。「その関係は、相手にとって『予定がないときの選択肢』です」。耳に痛いですが、事実です。友人整理の核心は、「呼んでくれる人」との時間を意図的に増やし、一方通行の関係には距離を置くことです。全員との関係を続けられる時間と心的余裕は、47歳には限られています。選別とは、残す人を決めることではなく、自分が呼ばれたい人の周りに圏を引くことです。

## 3. 「誠実さ」と「共通項」のバランスを保つ

関係を整理するとき、多くの人は「誠実さのない友人を外そう」と考えます。でも危険があります。誠実さだけを基準にすると、「正論パートナー」に囲まれた窮屈な人生になります。人生の重たい話を聴いてくれる友人も大切ですが、理由もなく一緒にいられる友人も同じくらい大切です。AIに「私にとって理想的な友人ポートフォリオは、どのような『バランス』ですか」と問うことで、友人を「カテゴリー化」できます。心の深さが必要な場面の1人、笑いが必要な場面の1人、沈黙が心地よい1人、成長を刺激してくれる1人。選別とは、全員を優先度1にしないことです。時間と心の使い分けの戦略です。

## 4. 「卒業する関係」の作法を学ぶ

友人を整理するとき、最難関が「関係の終わらせ方」です。突然距離を置けば、相手を傷つけます。かといって無理に続けば、自分が疲弊します。AIに「誠実に関係を卒業するための3段階」を聴くと、プロセスが見える化されます。第1段階は「返信速度を落とす」。反応が遅くなることで、相手も「優先度が下がった」と感じます。第2段階は「共通の場での会話を減らす」。グループLINEでの返信を控える。第3段階は「1対1の機会を作らない」。この3段階を半年かけて実行すれば、関係は自然に希薄になります。卒業は突然ではなく、波のような消退です。

## 体験談: 「全員と仲良くしたい病」を手放した話

僕は30代までの20年間、「誰とでも仲良くしたい」という呪いに生きていました。誘いがあれば断らない、相手が求めれば応じる、その結果、スケジュール帳は埋まっているのに、心は空だった。40代に入ったとき、同時に3つの関係トラブルが重なって、ようやく気づきました。「全員と深い関係は不可能」という当たり前の事実に。AIに「あなたが誰との時間を大事にしたいのか」を何度も問い直してもらった結果、整理が進みました。最初の3ヶ月は罪悪感がありました。返信しない自分に、距離を置く自分に、誘いを断る自分に。でも4ヶ月目、化学変化が起こりました。関係を整理した友人の誘いが、心から嬉しくなった。深さが違うのです。全員と仲良くしていたときの「楽しさ」と、選んだ誰かとの「楽しさ」は、別の感情だった。その差を感じた瞬間、47歳の人生が少しだけ軽くなりました。

## 🎯 上級プロンプト: 「友人の整理・選別」深掘り対話テンプレ

下記をAIに丸ごと貼り付けてください。「友人の整理・選別」に関する現状を構造的に問い直し、行動可能なネクストステップに翻訳します。

```
あなたは47歳の管理職に伴走するコーチです。私はいま「友人の整理・選別」というテーマで思考整理をしたい。
以下の順序で対話してください。
1. まず私に「友人の整理・選別に関して、直近1ヶ月で起きた具体的な出来事」を3つ尋ねる。
2. 私の答えから、その3つに共通する「私の癖・前提・思い込み」を3つ言語化する。
3. その癖を踏まえ、「次の30日で取り組む行動」を難易度★1〜★3で5つ提案する。
4. 最後に「3年後の私が今の私を見て泣くか笑うか」を一言で評価する。
```

このプロンプトは抽象的な悩みを「事実→癖→行動→未来評価」の4段階に分解します。1回で終わらせず、★1の行動を実装した翌週に同じプロンプトを再投入すると、変化が定量化されて自信に変わります。

## 📖 追加体験談・失敗回避リスト

47歳の私が「友人の整理・選別」で踏んだ3つの地雷を共有します。第一に、AIの回答をコピペして満足し、自分の言葉で語り直さなかったこと。半年後、何一つ身体に残りませんでした。第二に、初回のAI対話だけで結論を出してしまったこと。本当は3回目以降に核心が出てきます。第三に、家族や信頼できる部下に共有しなかったこと。共有しない学びは1ヶ月で蒸発します。失敗回避リストは下記。
