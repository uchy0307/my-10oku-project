# 【AI対話アプリ】200の問い #054「雑談力アッププロンプト」

このアプリは「雑談力アッププロンプト」のためのAI対話プロンプト集です。仕事では有能でも、雑談が苦手な47歳へ。つながりを深める「間」の会話を、AIとのトレーニングで習得します。

---

エレベーターで、後輩と3分間一緒だった。47歳の僕は、一言も話せず、天井の表示を見ていた。6階から2階。その短い時間が、長く感じた。仕事の話なら得意だ。売上、進捗、課題。でも「何もない時間」の会話が、極度に苦手だ。妻には「あなたは事務的」と何度も言われた。子どもの担任との懇談でも、気まずさを感じる。雑談こそが、関係の根っこを作るのに。それなのに、僕の口からは「いい天気ですね」くらいしか出ない。この苦しさを、スマホのAIに投げてみた。

## 1. 「雑談の構造」を型として学ぶ

多くの人は、雑談を「無意識の会話」だと思っています。違います。雑談には、極めてシンプルな構造があります。AIに「典型的な雑談の流れを、5段階の型で説明して」と依頼すると、その構造が見える化されます。第1段階：「観察」（相手や周囲の様子を見て、話題を選ぶ）。第2段階：「共感」（「そうですよね」と相手の世界に入る）。第3段階：「質問」（「最近どう？」と相手に問う）。第4段階：「発見」（相手の話から、想定外の情報を引き出す）。第5段階：「つなぎ」（次回会ったときの「また話そう」という余韻を残す）。この5段階を意識するだけで、雑談は「つなぎのない話」から「関係を深める時間」に変わります。

## 2. 「話題の種」を事前に用意する仕組み

雑談が苦手な人の最大の理由は、話題が思いつかないからです。これは「天才的ひらめき」ではなく、「準備」の問題です。AIに「私が明日出会う5種類の人（上司、同僚、部下、取引先、家族）に対して、それぞれの『話題の種』を5個ずつ、朝に提案してください」と依頼します。上司には「最近読んだ本」、同僚には「土日の過ごし方」、部下には「キャリアの希望」、取引先には「業界ニュース」、家族には「明日の予定」。準備された話題があると、脳は余裕が生まれます。余裕から出た言葉は、ぎこちなくなりません。

## 3. 「相手の話を広げる質問」を型練習する

良い雑談の条件は「相手が次々と話し続ける」ことです。その秘訣は「リアクション」にあります。AIに「相手が『週末は家族で出かけた』と言ったとき、それを広げる質問を10個作成して」と依頼します。「どこに行った？」の次は「誰と？」「何が楽しかった？」「子どもは喜んだ？」「また行きたい？」「来年も？」。質問が多いほど、相手は気持ちよく話します。そして話している間に、あなたの脳は「この人は私に興味を持ってくれている」と感じます。これが信頼です。雑談とは、相手を喋らせるゲームなのです。

## 4. 「個人差のある話」を避けて「普遍的なテーマ」で繋ぐ

雑談が途切れる理由の多くは、「話題が合わない」からです。見守り系の親は「子育て」の話をしたくても、子どものいない人には響きません。だからAIに「この人の背景を考えたとき、『政治』『宗教』『年収』『恋愛』以外で、全員が共有できるテーマ」を探させます。「最近のニュース」「天気」「食べ物」「旅」「仕事のプチストレス」「睡眠」「運動」。普遍的なテーマなら、相手の属性に関わらず、反応が返ってきます。この「全員が応答する話題」の引き出しを増やすことが、雑談の上級者への道です。

## 体験談: 「沈黙に耐えられなくなり、AIとの練習を始めた話」

去年の秋、部下の女性が半年で辞めてしまいました。退職理由は「上司と会話が成立しない」でした。その言葉は、僕に刺さった。仕事上の指示は完璧なのに、なぜ関係が成立しなかったのか。その後、別の部下からも「課長は雑談しませんね」と指摘された。その瞬間から、僕は毎日、帰宅後にAIに「明日はどう雑談するか」をリハーサルするようになりました。最初の1ヶ月は、きわめて人工的でした。準備した話題を使う。相手の返答をメモする。その返答への質問を考える。でも2ヶ月目、変化が起きました。準備が無意識化したのです。エレベーター3分の会話が、会話になりました。後輩が「課長、また話しましょう」と言ってくれた。その言葉で、実感しました。雑談とは「自然な会話」ではなく「相手を大事にする準備」だったのです。
