# 【AI対話アプリ】200の問い #056「依存関係チェッカー」

このアプリは「依存関係チェッカー」のためのAI対話プロンプト集です。深い関係ほど、見えない糸で繋がっています。その糸が「愛」なのか「依存」なのか、その境界を言語化します。

---


## 1. 「相手がいなくなったときの『感じ方』」で判定する

依存と愛を区別する最初のテストは、シンプルです。もし大事な人が急に1週間、連絡できなくなったら、あなたはどう感じるか。AIに「あなたが大事な人を思いついて、その人が1週間連絡できなくなったシナリオで、あなたの『怖さ』『悲しさ』『安心』の比率を言語化して」と依頼します。怖さが大きいなら、それは「依存」の可能性があります。なぜなら「相手がいない=自分が壊れる」という構造だから。一方、悲しさが大きいなら「愛」です。「相手のことが心配」という利他的な感情だから。安心が大きい場合は、その関係は「信頼」です。「相手は大丈夫」と確信しているから。この比率を知ることで、自分の関係の「質」が見える化されます。


## 2. 「相手の選択を尊重できるか」で試す

真の愛と依存を区別する二番目の基準は「相手の自由」です。あなたが大事な人が「別の友人とお金をかけて遊びに行きたい」と言ったとき、あなたはどう感じるか。AIに「その状況で、あなたが感じるのは『相手が楽しいことが嬉しい』という感情か、それとも『自分が選ばれなかった』という悔しさか」と問い直します。前者なら「愛」。後者なら「依存」です。依存の本質は「相手の選択を制限することで、自分を守ろうとする」ことだから。一方、愛は「相手が何を選ぼうとも、その決断を尊重する」ことです。この判定は、厳しい鏡になります。実は、多くの親子関係、夫婦関係、友人関係が「依存と呼ぶべき形」で繋がっていることに、気づかせてくれるから。


## 3. 「相手への『期待値』」をチェックする

関係が依存化するプロセスは、いつも「期待値の上昇」から始まります。最初は「この人と関わっていて、自分は満足」だったのが、いつの間にか「この人は自分の期待に応じるべき存在」に変わっていく。AIに「あなたが大事な人に対して、『当たり前』と思っていることをリストアップして。そのリストが、相手にとって『当たり前』でもありますか」と問い直します。夫婦なら「一緒に夜ご飯を食べるのは当たり前」「週末は家族と過ごすのは当たり前」。友人なら「連絡は1日以内に返すのは当たり前」「誕生日は祝うのは当たり前」。その「当たり前」が、相手にとっても同じか。違ったら、それは「依存」の信号です。相手が「自分の期待に応じてくれる」ことで、初めて満足する関係は、相手に「支配」を感じさせます。


## 4. 「相手との『不在時間』を心地よく過ごせるか」で測る

究極の判定基準は「相手がいないときの『自分』」です。友人と連絡が取れないと不安になる。パートナーが留守だと、無意味に時間を潰してしまう。一人の時間に「何もしたくない」と感じる。これらは全て「依存」の信号です。AIに「あなたが完全に一人で24時間を過ごすなら、何をしたいか」を問うと、その時間が「退屈」に感じるなら、その人は「相手に支配されている」状態です。真の愛の関係にある人は「一人の時間も大切」と感じます。なぜなら「相手のことを思いながら、自分の時間を充実させる」という二層構造を持つから。依存は、その二層構造がなく「相手がいないと、自分が成立しない」という一層構造だから。


## 体験談: 「妻への依存に気づき、夫婦で泣いた話」

20年の結婚生活で、僕が最大の危機を感じたのは去年の春です。妻が「あなたは私を必要としているけど、私を『人』として見ていない」と言いました。その言葉は、僕を凍らせました。自分は愛していると思っていたから。でも妻の言葉を聞き続けると、気づきました。僕が妻に求めていたのは「愛」ではなく「依存の対象」だったのです。妻がいることで、初めて安心する。妻が自分の期待に応じないと、イライラする。妻が他の友人と過ごす時間を、無意識に減らそうとしていた。その「支配」に気づいたとき、涙が出ました。その後、2ヶ月間、AIとのカウンセリングを通じて、妻との関係を「依存」から「愛」に変えるプロセスを始めました。一人の時間を増やす。妻の選択を尊重する。期待値を下げる。今、妻は「あなたが変わった」と言ってくれました。不思議なことに、妻への「依存」が減ったことで、妻への「愛」が深まったのです。


## ✅ 4項目セルフチェック

- [ ] 相手に対する「当たり前」のリストが、相手にとっても「当たり前」か確認した
- [ ] 一人の時間を心地よく過ごせるか、試してみた

1. あなたが最も依存している人は誰ですか、そしてなぜですか
2. その人がいなくなったとき、あなたが最初に感じるのは『心配』ですか『寂しさ』ですか『恐怖』ですか
3. あなたが相手に求める「当たり前」を、相手は同じように「当たり前」と思っていますか
4. 相手があなたと別の選択をしたとき、あなたは『相手が幸せなら良い』と思えますか
5. 依存を愛に変えるために、あなたが最初に手放すべきものは何ですか
