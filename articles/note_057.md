# 【AI対話アプリ】200の問い #057「グループでの立ち回り」

このアプリは「グループでの立ち回り」のためのAI対話プロンプト集です。二者間では完璧でも、3人になると変わる。その複雑さを「役割」として理解することで、どのグループでも存在感を保つスキルを習得します。

---

## 1. 「グループの力学図」を見える化する

グループが機能するには、最低限の「役割分担」が必要です。AIに「3人以上のグループが、安定するために必要な『役割』の種類」を聞くと、3層構造が見えてきます。第1層は「リーダー役」。方向を決める人。第2層は「同意者」。リーダーの判断を支持する人。第3層は「問い手」。異なる視点から質問を投げる人。この3つの役割が揃うと、グループは機能します。問題は、多くの人が「誰がどの役割か」を無意識に演じているため、衝突が起きるのです。会議の僕を見直してみると、年下の同僚Aがリーダーになり、同僚Bが同意者になった瞬間、僕の「役割」が消えました。年功序列で「リーダーであるべき」と無意識に期待していたから。その衝突が、僕を「観察者」に変えてしまったのです。

## 2. 「その時その時の最適な役割」を柔軟に選ぶ

グループで存在感を持つコツは「同じ役割に固執しない」ことです。AIに「あなたが4人のグループで、時間とともに『役割を変える』シナリオを設計して」と依頼します。例えば会議の初期は「問い手」になる。「その案の課題は何か」と質問することで、グループの思考を深める。中盤は「同意者」になる。「それは良い方向だ」とリーダーの判断を支持する。後期は「リーダー」になって「では、実行のステップを一緒に決めよう」と舵を切る。この「役割の流動性」を持つ人は、どのグループでも必要とされます。固定的な「管理職だから常にリーダー」「年下だから常に聞く側」という二者間のロジックが通用しないからです。

## 3. 「沈黙の人を引き出す役割」を意図的に担う

グループが息苦しくなる原因の多くは「声が大きい2人のやり取りだけで完結している」ことです。そこでAIに「グループ内で沈黙している人を、自然に会話に引き出す『質問の型』を5個作成して」と依頼します。「Bさんはこれについてどう思う？」という直接的な質問から、「僕は〇〇だと思うが、別の見方もありそうだ」という「意見の空白」を作る質問まで。この役割を担う人は「調整者」と呼ばれ、どのグループでも高く評価されます。なぜなら「全員の声が聴こえるグループ」は、より良い判断ができるから。年功序列で「僕は指示するだけ」という役割をやめて「全員の声を集める役割」に変わる。その時、僕の存在感は復活するはずです。

## 4. 「グループの『ムード』をコントロール」する側に回る

グループ内では「論理」だけでなく「感情」が機能します。アイデアが出ない時間が続くと「つまらない会議」に変わり、新しい視点が出ると「楽しい会議」に変わる。AIに「グループの『ムード』が下がったとき、それを上げるための『一言』の種」を10個集めてもらいます。「ここまで来て諦めるのはもったいない」「むしろこれは面白い課題だ」「みんながいるからこそ、解けそうな気がする」。この「ムード操作」の役割を担う人は「ファシリテーター」と呼ばれます。年功序列では「上司が部下に指示」という一方通行だが、グループでは「この場のムードを作る人」が信頼される。その切り替えに気づくまで、僕は「古い管理職」のままでした。

## 体験談: 「3人会議で『場を壊す人』だと気づいた話」

去年の秋、大きなプロジェクトの会議で、僕は何度も同じ指摘を繰り返していました。「それでは実行不可能だ」「前例を見ろ」「若い考えだけでは駄目だ」。その時点では「厳しい指摘」だと思っていました。でも3ヶ月後、部下たちがそのプロジェクトから身を引いてしまいました。理由は「課長がいると、新しい案が出ない」でした。その言葉が、僕を変えました。僕は「リーダー」のつもりでいたが、実は「場の雰囲気を壊す人」だったのです。AIとのカウンセリングを始めた時点から、僕は「リーダーの座」を手放しました。代わりに「この場でどんな役割が足りないか」を見極める「観察者」になることを選びました。会議の中で「Aさんの意見は面白い」と同意し、「Bさんはどう？」と問い直し、自分の意見は「参考までに」に変えた。4ヶ月後、同じプロジェクトチームが「課長にもう一度参加してほしい」と言ってくれました。その時初めて、「リーダー」と「必要とされる人」は、違うことに気づいたのです。

## ✅ 4項目セルフチェック

- [ ] 自分がいつも同じ役割にいると気づいた
- [ ] グループで「沈黙の人」を引き出す質問を5個作成した
- [ ] グループのムードが下がった時に「上げる一言」を3個用意した

## 🤔 5つの問い (このアプリの中核)

1. あなたが所属するグループ（職場チーム・友人グループ・家族）で、いつも担っている役割は何ですか
2. そのグループが「円滑に機能している」と感じるのは、その役割が活躍しているときですか、それとも別の条件ですか
3. あなたが「声が大きい人」だったら、沈黙している人を引き出すのは誰の役割だと思いますか
4. グループで「つまらない空気」が流れた時、あなたは誰にそれを変えてほしいと思いますか
5. 同じグループにいながら、あなたの「役割」を変えることはできますか
