# 【AI対話アプリ】200の問い #059「褒め言葉のバリエーション」

このアプリは「褒め言葉のバリエーション」のためのAI対話プロンプト集です。「いいね」「素晴らしい」の単語化した褒め方では、相手の心に届きません。その人の本質を言語化する褒め言葉の引き出しを、100の角度から学びます。

部下が企画書を提出した。47歳の僕は「いいね」と返しました。部下の顔が、少し落ちた。僕は疑問に思いながらも「頑張ったな」と続けました。それでも表情は変わらない。その時、妻の言葉がよみがえってきました。「あなたの褒め方は、心がこもっていない」。実は、多くの場面で同じことが起きています。子どもに「良い成績だね」と言い、妻に「美しいね」と言う。でも、相手の反応は薄い。それは、褒めが「一般的」だからです。「いいね」は誰にでも言える。その人の「何が」「どうして」「なぜ」素晴らしいのかを言語化できれば、褒めは「相手を見ている」というメッセージに変わります。スマホを取り出し、AIに「その人の本質を褒める言葉の作り方」を問いました。

## 1. 「褒める前に、相手を『観察』する」プロセス

褒め言葉が浅くなる理由は、相手を十分に観察していないからです。AIに「ある人を褒めるとき、その人の『何を見て』褒めるべきか」と問うと、観察の層が見える化されます。第1層：「行動」を見る。「提案書をまとめてくれた」。第2層：「工夫」を見る。「3人の意見を統合する工夫が見える」。第3層：「姿勢」を見る。「チーム全体の成功を最優先にする姿勢」。第4層：「本質」を見る。「困難な状況でも、前に進もうとする勇気を持っている」。単なる「いいね」は第1層。「工夫が見えて素晴らしい」で第2層。「そういう姿勢だからこそ、チームが信頼する」で第3層。「その勇気があるから、他の人も動く」で第4層。この階層的な観察を通すと、褒め言葉は「相手の本質を見ている」というメッセージに変わります。

## 2. 「その人の『弱さ』を褒める発見」

褒め言葉で最も響くのは、実は「短所を肯定的に言い換える」ことです。AIに「ある人の『欠点』と『それが実は強みになっている場面』を対になるように言語化する」と依頼します。例えば「完璧を求める人」は「細かい」と短所に見えますが「品質へのこだわりがあるから、チームの信頼度が高い」と言い換えられます。「落ち着きのない人」は「浮気っぽい」に見えますが「好奇心が強いから、新しい課題に果敢に挑戦できる」に変わります。この「短所の肯定的再解釈」が、最も相手の心に刺さります。なぜなら「あなたを短所ごと理解して、肯定している」というメッセージだから。

## 3. 「特定の場面」と「その人の選択」を繋ぎ合わせて褒める

褒めが響かない理由の多くは「一般的」だからです。AIに「ある人の『特定の判断』について、その判断の背景にある『その人の価値観』を推測しながら褒める」と依頼します。例えば部下が「自分で調べるのに3時間かかる提案を、チームに相談して1時間で完成させた」という場面。単なる褒めは「いいね」。でも背景まで見ると「自分の時間より、チームの成功を優先する価値観を持っている」と推測できます。その推測を言葉にすると「君が相談を選んだのは、仕事を効率化するためじゃなくて、チームが最高の成果を上げることを最優先にしているからだと思う。その優先順位の付け方が、本当にいい」という褒め言葉になります。

## 4. 「褒め言葉の『具体性』を深める工夫」

褒めが心に届かない最大の理由は「相手が信じられないから」です。AIに「その人が『これは褒めているが、本当か』と疑うような褒め言葉」を意図的に作ると、その人が本当に褒めてほしいポイントが見えてきます。例えば部下が「こんなの大したことない」と言う褒めがあれば、それはその人の心に届いていない。そこで「君のプレゼンの何が素晴らしいか、具体的に3つ挙げよう。第1に、難しい概念を『子どもにも分かる例』に置き換える工夫。第2に、聴き手の『質問が出そうな場所』を先読みして説明している。第3に、最後に『では、ここから何をすべきか』という行動レベルまで落とし込んでいる」と具体化すると、部下は「この人は本当に見ている」と感じます。

## 体験談: 「褒め言葉で、距離があった部下が泣いてくれた話」

2年前、僕の部下で「内向的」と評されている女性がいました。会議では発言しない。求められた仕事は完璧にこなすが「何を考えているか分からない」と評価され、昇進候補から外されていました。ある日、その部下の業務フローを見つめ直して気づきました。彼女は「誰も気づかない細かい課題」を、ずっと一人で解決しつづけていたのです。その「気づかれない工夫」の積み重ねが、チーム全体の効率化を支えていました。その日、僕は彼女を呼んで、30分かけて「君が何をしているか、何を大事にしているか」を言語化しました。「君は、目立つ成果よりも『気づかれない課題の解決』を優先している。その姿勢は、派手に見えないが、チームの『根っこ』を支えている」。その言葉を聞いた時、彼女の目に涙がたまった。その後、彼女は「ようやく、自分が見られている気がした」と言ってくれました。昇進を勧めたわけではない。ただ「その人の本質を褒めた」だけで、彼女の人生観が変わったのです。
