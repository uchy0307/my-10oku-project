# 【AI対話アプリ】200の問い #061「爆速タスク整理」

このアプリは「爆速タスク整理」のためのAI対話プロンプト集です。頭の中に散在している20個のタスクを、優先度・実行コスト・インパクトの3軸で5分で仕分け、今月やることと寝かすことが明確になります。

---

## 1. 「締切」「インパクト」「コスト」で座標軸を作る

タスク管理の最大の課題は、「重要度」という言葉が曖昧なこと。AIに「私のタスクを3軸で整理して: X軸=締切の迫り度(1-10) Y軸=会社への影響度(1-10) Z軸=実行に必要な時間(1-5時間)」と告げると、脳内で湯気が立つほど整理される。座標軸を作った瞬間、「月末までに終わらせる」と「心理的に後回しにしていた」が別物だと気づく。私の場合、「新入社員育成」は締切はないのに心理的重みが大きく、実は「月末報告書」が締切まで1週間で最優先であることが浮かぶ。座標軸は単なる分類ではなく、思考の盲点を照らす懐中電灯です。

## 2. 「今日中・今週中・今月中」で3段階の時間枠を設定

タスク管理で次に起こるのは、「やることはわかった、では何からやる?」という問題。AIに「3軸の座標軸で優先度が高いタスク5個について、実行期間を『今日中』『今週中』『今月中』のいずれかに割り当ててください」と指示すると、脳内の熱量がタスクに流れ込む。「今日中」に割り当てられたタスクは、朝から19時までのうち、どの時間帯に充てるかが明確になる。最初の3つの時間枠に頭をリセットさせることで、「来週でいいかな」という先延ばしが物理的に不可能になるのです。

## 3. 1タスク最大3時間、それ以上は「分割」を強制

タスクが完結しない理由の多くは、「5時間かかるタスク」を無理やり1日でやろうとするから。AIに「実行コストが5時間以上のタスクについて、『1-3時間で完結するサブタスク』に自動で分割してください」と頼むと、一気に実現可能性が上がる。「部門予算の再集計」は5時間かかるため、①前年度データの整理(1時間)、②部門別の修正見積もり(1.5時間)、③エクセル記入と検算(1.5時間)、に分割される。1日3時間の制約の中で、1つのサブタスクなら確実に終わる手応えが生まれます。

## 体験談: 「締切」「インパクト」「コスト」で座標軸を作る

実は私、1年前までは毎朝、このタスク優先度判定に2時間を費やしていました。ノートに書き出し、消し、また書き出し。妻から「なぜそんなに長いの?」と聞かれても答えられない。その時間帯は、実は「判断そのもの」で脳が疲れていたのです。AIを導入した当初、「座標軸で仕分けしてください」と指示したら、出力は60秒でした。私が2時間悩んでいたことを、AIは座標軸という「ルール」で即座に処理する。その瞬間、「優先度判定」という作業が、実は「判断ルール」の問題だったことに気づきました。今では月曜朝の8時10分には、「今週やること・来週に回すこと・今月中にやること」がA4一枚に整理されている。朝8時半から、脳は本来の仕事に集中できる。この5分の効率化は、月50時間の判断疲れを削ります。

## ✅ 4項目セルフチェック

- [ ] 自分のタスク15個を「締切・影響度・コスト」の3軸座標に落とし込んだ
- [ ] 「今日中」「今週中」「今月中」の3時間枠にタスクを割り当てた
- [ ] 実行コストが3時間を超えるタスクを、サブタスクに分割した
- [ ] 「捨てるタスク」を2個、明確に指定した

## 🤔 5つの問い (このアプリの中核)

1. あなたが「今日中にやらなければ」と焦っているタスクは、本当に今日が締切ですか
2. そのタスクを完了した時、あなたの会社にもたらされる影響は、本当に大きいですか
3. 今、タスク判定に時間をかけているあなたを見て、3年前の自分は何と言いますか
4. もし「今月捨てるべきタスク」が3個あるなら、どれですか、そして理由は
5. 月末、すべてのタスクが完了した状態と、80%完了でも心が満たされた状態、あなたはどちらを選びますか
