# 【AI対話アプリ】200の問い #065「アイデア量産・発想法」

このアプリは「アイデア量産・発想法」のためのAI対話プロンプト集です。『いいアイデアが思いつかない』を、創造的思考の3つのセットで『月30個のアイデア』に変える学習プロンプト群です。

## 1. 「0→1」ではなく「既存×既存→新結合」で考える

アイデアが出ない理由は、『全く新しいもの』を想像しようとするから。AIに「あなたの業界で『既に存在する概念・商品・サービス』を5つ挙げてください。その5つを『組み合わせたら』何が生まれますか?」と投げかけます。ビジネスの99%のアイデアは『既存の組み合わせ』です。営業手法と教育プログラムを組み合わせたら『営業研修』。オンラインと診断を組み合わせたら『オンライン診断』。組み合わせるだけで『新しい』が生まれる。


## 2. 「このシーン、困ってることは?」で市場を観察する

アイデアの源泉は『困っている人』の観察です。AIに「あなたが見かける『ちょっと困っている場面』を、この1週間で5つ思い出してください。その『困りごと』を、ITで解決したら何が作れますか?」と依頼します。妻の『毎日の朝食メニュー決定に5分かかる』→『朝食メニュー提案アプリ』。部下の『報告資料の様式を毎回確認している』→『報告資料テンプレート共有ツール』。


## 3. 月1回、『制約条件アイデア出し』で創造筋を鍛える

アイデアは『無制限』だと出にくく、『制約』があると爆発します。AIに『あなたの会社で『今月のテーマ:〇〇を解決するアイデア、ただし予算は1万円、期間は1ヶ月、人員は2名』という制約条件を与えて、30個アイデアを出し、その中で『明日実行できる』3つを選んでください』と頼みます。制約の中で考えるクセがつくと、月30個のアイデアが『月3個の実装案』に変わります。


## 体験談: 「0→1」ではなく「既存×既存→新結合」で考える

5年前、企画会議で『何かいいアイデアない?』と上司に聞かれ、私は黙ってしまいました。その時の悔しさが忘れられません。以来、『アイデア出しのコツ』を探してきました。参考書も5冊読みました。でも『ひらめき』は来ませんでした。転機は『アイデアは組み合わせ』という概念に気づいた時です。営業ツールと顧客教育の『組み合わせ』から『営業支援研修プログラム』が生まれ、それが会社で月3件受注できる商材になりました。それ以降、月30個のアイデアは『既存の組み合わせ』から出すようになり、その中から『実装可能な3個』を毎月選びます。創造性は『ひらめき』ではなく『仕組み』だったのです。

大量のアイデアの中から『本当に実行すべき3つ』を選ぶ際は、AIに『時間・お金・心理的ハードルの低さ』で並べ替えさせます。一晩で50個出したアイデアも、優先順位がつかなければ未着手のまま埋もれる。AIによる優先付けがあれば、明日の朝から具体的な一歩を踏み出せる。アイデアは、量と並び替えの両輪で価値が確定するのです。


## ✅ 4項目セルフチェック

- [ ] 今月、自社に『既存する概念・商品』5つを挙げた
- [ ] その5つの『新しい組み合わせ』から3つ新しい企画を考えた
- [ ] この1週間で『困っている場面』5つを観察し、解決アイデアを出した
- [ ] 『制約条件:予算1万円、期間1ヶ月』でアイデアを30個出し、3つ選んだ


## 🤔 5つの問い (このアプリの中核)

1. あなたが『いいアイデアだ』と感じる基準は何ですか、『実装可能性』ですか、『新規性』ですか
2. アイデアが『思いつかない』のは、思考の問題ですか、『観察不足』の問題ですか
3. 『既存のものを組み合わせる』と『全く新しいものを発明する』は、難易度が同じだと思いますか
4. もし『月30個のアイデアから、月2個が市場で受け入れられる』なら、あなたは何年で事業化できますか
5. アイデア出しが『上手い人』と『下手な人』の違いを見たとき、それは『才能』ですか、『習慣』ですか
