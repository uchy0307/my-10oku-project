# 【AI対話アプリ】200の問い #109「相続・家族対話準備」

このアプリは「相続・家族対話準備」のためのAI対話プロンプト集です。親の健康不安が増す時期に、資産・介護・見守りについて「今から家族で何を話すべきか」を構造化し、トラブルを避けるための対話シナリオを用意します。

---


## 1. 「自分が知らない親の資産」を明確にする

「自分が知らない親の資産」を明確にする

親の健康に不安を感じ始める頃、漠然と頭をよぎるのは「もしも」の時のこと。その中でも、親の資産状況は、なかなか正面から尋ねにくいテーマではないでしょうか。預貯金、不動産、有価証券、保険、年金、デジタル資産、さらには貸金庫の中身まで、その種類は多岐にわたります。私がそうであったように、多くの方が「実家には何かあるはずだが、具体的に何が、どこに、どれくらいあるのか全く分からない」という状態かもしれません。この「知らない」という状況こそが、将来の家族間の誤解やトラブルの種になりかねないのです。

このセクションでは、AIが提供する問いかけを通じて、漠然とした不安を具体的な情報へと変えていく手助けをします。例えば、「親の保有する不動産について、所在地や名義、購入時期などをどのように尋ねれば良いですか？」や「親が利用している金融機関や証券会社を把握するための具体的な質問例を教えてください。」といった問いかけは、対話のきっかけを明確にし、親御さんが話しやすい環境を整える上で非常に有効です。私がこのアプリを使う中で、漠然とした心配が整理され、建設的な対話への一歩を踏み出せたように、あなたも安心して親御さんとの対話に臨めるようになるでしょう。この準備が、将来の家族の平穏につながります。


## 2. 「介護費用」の現実を数値化する

多くの家族は「親が要介護になったら施設に入れる」と漠然と考えていますが、月々30-50万かかる現実を知りません。AIに「あなたの親が『要介護3』になった場合、月の介護費用がいくらか、市価を調べた上で『親の資産で何年持つか』を計算して」と頼みます。年金が月20万、貯金が1000万なら、介護費用月40万だと「25ヶ月で貯金が尽きる」という現実が見えます。その時点で「いつ誰が経済的にサポートするのか」という家族対話が必須になります。


## 3. 「親が亡くなったときの手続き」を事前リスト化する

親が亡くなった直後の3ヶ月は「やることリスト」が100個以上あります。死亡届、銀行口座凍結、健康保険喪失手続き、相続税申告、遺産分割協議——どれを後回しにするか判断を誤ると、相続がこじれます。AIに「あなたの親が亡くなった直後1ヶ月にやるべき手続きを『期限付き』で詳細に列挙して。その上で『誰が何をするか』を家族で分担する案」を作ってもらいます。事前にそのリストを親に見せて「このプロセスで、できれば遺言書をこの様式で作っておいてほしい」と伝えると、親も「自分の人生の終わり方」を具体的に考えるようになります。


## 4. 「兄弟姉妹の思いやり」をAIが可視化する

相続が揉める理由の50%は「金銭」ではなく「心情」です。兄は仕事で忙しい、妹は親の近くにいた、弟は親の介護をしていない——そういう「不公平感」が相続額の不満に化けます。AIに「あなたの兄弟姉妹の立場から、それぞれが『親の相続に対して何を期待しているか』『何に不満を持つリスクがあるか』を推測して」と頼みます。その推測を基に「相続前に『なぜこの配分にするのか』という親からのメッセージ」を親に書かせると、トラブルが激減します。


## 体験談: 「親が入院した3日で」家族会議が変わった話

正直に書きます。父の入院から1週間、私は病院と仕事の往復で追い詰められていました。兄からは「親の資産どうなってるの」という連絡。妹からは「介護の話もしないと」という不安。3人で全く情報が異なっていました。7日目、AI対話を始めました。まず「親の資産を知るための質問シナリオ10個」が出てきました。それを手に、父のベッド脇で、看護師の指示を待つ間に「父さん、銀行口座どこにある?」と聞きました。最初、父は答えたくなかったようです。でも「相続をスムーズにするため」と説明すると、事細かに教えてくれました。その情報を兄妹にLINEで共有しました。翌日、「介護費用の月額」をAIに計算してもらいました。月45万、親の年金月18万、月27万の赤字。5年続くと1620万必要。親の預貯金は2000万。「あと8年持つ」という数字が出て、初めて家族で「現実的な介護計画」の会話が始まりました。10日目、「亡くなった直後の手続きリスト」を作りました。そのリストを親に見せながら「この手続きで遺言書が必須。この形式で書いてほしい」と伝えました。父は「こんなに手続きがあるんだ」と驚き、その週中に遺言書を弁護士に依頼しました。今、父の容体は安定しています。でも「この3日のAI対話がなかったら」と思うと、後々の家族紛争が目に見えていました。

「親が『遺言書を書く決心がつかない』ときの最終手段。あなたが親に宛てた『あなたの人生への感謝とこれからの約束』の手紙を、親に読ませる前にAIに添削してもらう。その手紙が、親の『遺産の配分をちゃんと決めよう』という決心につながる」

```
あなたは家族間のコミュニケーションを円滑にする専門コーチであり、相続、介護、見守りに関する家族会議の準備をサポートします。これから、家族との対話に向けたあなたの考えを整理し、具体的な行動計画を立てるお手伝いをします。

まず、あなたの家族において、親の資産、介護、見守りに関して、現在どのような状況にあるか、またはどのような漠然とした不安を感じていますか？ 具体的な情報がなくても構いませんので、現状の認識や懸念を教えてください。

次に、家族でこれらのテーマについて話し合うことで、どのような状態を目指したいですか？ また、対話を始める上で、どのような点が障壁になる可能性があると思いますか？ 例えば、家族の意見の相違や親の抵抗などが考えられます。

これらの課題を解決し、目指す状態に近づくために、事前にどのような情報（例: 親の意向、現在の資産状況、利用可能な介護サービス、かかりつけ医の情報など）を収集しておくべきだと考えますか？ また、それらの情報をどのように収集できそうでしょうか。

最後に、これらの情報を踏まえ、家族会議をスムーズに進めるために、どのようなアプローチが考えられますか？ 参加者、話し合う場所、議題の提示方法など、具体的なアイデアがあれば教えてください。

あなたの回答に基づいて、具体的な次の一歩を三つ提案します。
```
