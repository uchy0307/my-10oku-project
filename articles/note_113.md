# 【AI対話アプリ】200の問い #113「デザイナー・コンセプト可視化」

このアプリは「デザイナー・コンセプト可視化」のためのAI対話プロンプト集です。デザイナーが「直感」で創作する背景にある「コンセプト」を言語化し、クライアントとの認識ズレをなくし、制作プロセスを透視化します。

---


## 1. 「デザインの『魂』を1文で定義する」

デザインの「魂」とは、その作品の根源にあるコンセプトであり、作り手の直感の源泉です。それをたった1文で定義する。これは、一見不可能に思えるかもしれませんが、実はクライアントとの認識ズレを防ぎ、制作プロセスを明確にする上で極めて重要です。

私自身、かつては直感を頼りにデザインを進め、クライアントとの最終的なズレに悩んだ経験があります。しかし、この「1文定義」を実践するようになってから、漠然とした「良い感じ」が、明確な意図を持つ「力強い表現」へと変わっていくのを実感しました。言葉にできない直感を、試行錯誤しながらも1文に凝縮することで、デザインの芯が際立ち、私自身の思考も驚くほどクリアになったのです。

AIへの問いかけ例としては、「このデザインの核となる『なぜ』を、誰にでも伝わるように一文で表現してください。」や「このデザインが最終的に達成すべきことを、キャッチーな一言で言い切るとしたら？」などがあります。この作業を通じて、あなたのデザインの真髄が言語化され、クライアントとの円滑なコミュニケーション、そして透明性の高い制作へと繋がっていくことでしょう。


## 2. 「3色の配色理由」を言語化する

デザインの80%は「色選択」で決まります。AIに「あなたが選んだメインカラー『何のために』『どの感情を表現するために』選びましたか。その色が『見る人に何を感じさせたいか』。同時に『サブカラー2色』も『その3色の関係性』『なぜこの3色の組み合わせなのか』を言語化してください」と頼みます。色は「好きだから」ではなく「心理学的に『青は信頼を、オレンジは親しみやすさを、白は透明性を表現する』から『高齢者に信頼感と親しみやすさを与える』という目的に達する」という『色の三角形』が見えてきます。


## 3. 「ターゴ人物の『朝の行動』から逆算設計する」

見る人・使う人の「具体的なシーン」から逆算してデザインします。AIに「あなたのデザインを『実際に見る・使う人』の『朝の行動』を想像してください。その人は何時に起きて、何を見て、何をしたいですか。その『3分間のシーン』の中で『あなたのデザインに出会う瞬間』『そこで何を感じるべきか』を描写してください」と頼みます。高齢の母が朝6時に起きて、老眼鏡で小さいボタンが見えず困っている。その瞬間に「私のパネルデザイン」に出会う。見やすさと温かみが同時に伝わる必要がある。そういう「1シーンの物語」から逆算すると、デザイン要素すべてが意味を持ちます。


## 4. 「競合との『違い』を『コンセプト差』で説明する」

クライアントから「競合と何が違うのか」と聞かれたとき、デザイナーは困ります。AIに「あなたのデザインと『競合の似たようなデザイン』を比較して『コンセプトレベルで何が違うのか』を3点、言語化してください。『色が違う』ではなく『何のために色が違うのか』『その色が誰にどう響くのか』という意図レベルの説明」を作成してもらいます。「私の青は『信頼感』、競合の青は『クール感』。ターゴの高齢者には『クール感』より『信頼感』が必要」——こういう『意図差』が言語化されると、ポートフォリオの説得力が激変します。


## 体験談: 「直感のデザイン」が「意図のデザイン」に変わった3週間

正直に書きます。デザイナー人生20年間「色に迷ったら直感」「形に迷ったら何度も試す」という手探りの方法をとってきました。それで「いいデザイン」は作れました。でも「なぜこれがいいのか」は説明できませんでした。初日のAI対話で「デザインの魂を1文で定義する」を求められたとき、初めて「高齢者が迷わない直感性」と「若々しさ」という矛盾を『同時に実現する』というコンセプトが見えました。その瞬間「なぜ大きなボタンなのか」「なぜこの色なのか」に根拠が生まれました。2日目「3色の配色理由」を言語化したとき、色彩心理学から逆算して「なぜこの組み合わせなのか」が説明できるようになりました。3日目「ターゴの朝の3分間」を描写したとき、デザインの全要素が『那珂市のシーンの中での役割』として意味を持ち始めました。1週目「競合との違い」を『コンセプトレベル』で言語化したとき、初めて「私のデザインは『信頼感をターゴに与える』もので、競合はそれを狙っていない」という『意図差』が見えました。翌週、クライアントに「このパネルデザインの意図」を説明したとき、クライアントが「ああ、だからこの色なんですね」と初めて納得してくれました。それ以来、修正指示が『的確』になり「デザイナーの意図を理解した上での提案』に変わりました。20年間の『直感のデザイン』が、『意図のデザイン』に昇華されました。

```

これらの問いへの回答を踏まえ、あなたのデザインコンセプトを明確にし、クライアントとの認識ズレをなくすための具体的な次の一歩を3つ提案してください。
```
