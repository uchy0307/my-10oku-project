# 【AI対話アプリ】200の問い #114「役作り・キャラクター分析」

## 1. 「その人物の『20年前』『今』『20年後』を3点で定義する」

役作りにおいて、目の前の台詞や行動だけでなく、その人物の深層心理や人生の背景を理解することは不可欠です。この「20年前」「今」「20年後」という三つの視点を用いることで、キャラクターの人生全体を俯瞰し、その人物がなぜ今そこにいるのか、何を感じているのかを立体的に構築します。

「20年前」では、その人物がまだ若かりし頃、どのような環境で育ち、どんな経験が価値観や性格の基礎を築いたのかを探ります。どのような夢を抱き、何に喜び、何に傷ついてきたのでしょうか。それが現在のパーソナリティにどう影響しているのかを考えます。「今」は、20年前の経験を経て、現在の状況や直面している課題、抱えている感情、そして目標を具体的に定義します。過去の経験が現在の選択や行動にどう結びついているのかを明確にします。そして「20年後」。現在の行動や選択がもたらすであろう未来を想像します。その人物が望む未来とは何か、何を成し遂げているのか、あるいは何が欠けているのか。未来の姿を逆算することで、現在の動機や葛藤がより鮮明になります。

AIへの問いかけ例としては、「この人物が20年前、一番大切にしていたものは何でしたか？それが現在の彼/彼女にどう影響していますか？」や、「もし現在の状況がこのまま20年続いたとしたら、彼は/彼女はどのような人物になっているでしょうか？その変化の根拠も教えてください。」などが有効です。

私自身、この手法を取り入れてから、演じる役への理解度が格段に深まりました。台詞の裏にある感情や、舞台上の瞬間の選択が、その人物の人生全体の文脈から自然と湧き出てくるような感覚を覚えました。それは単なる演技ではなく、その人物として生きる体験でした。この分析は、キャラクターの内的論理を構築する上で強力なツールとなるでしょう。


## 2. 「『台本の場面』で、その人物が『本当に言いたいこと』を見つける」

台詞は「セリフの表面」と「本当の感情」が異なります。AIに「舞台のこのシーンで上司が『新入社員に教える』という台詞がありますが『本当のところ、その上司は何を伝えたいのか』『その台詞の『下層』に何があるのか』を『その上司の内的独白』として書いてください」と頼みます。「がんばれ」という台詞の下層には「俺みたいになるな」という無意識の願いがあるかもしれません。その『本当の感情』が舞台上の一言一言の『質感』を変えます。


## 3. 「『その人物が恐れていること』『その人物が望んでいること』を二者択一で言語化する」

キャラクターは「矛盾」で生きています。AIに「この上司が『心の奥底で恐れていることは何ですか』『その恐れから逃げるために『何を成し遂げたいと思っていますか』を『その上司の目線』で『今この瞬間』に書いてください」と頼みます。恐れは「人生の失敗」「存在の空虚感」「定年後の無用感」。その恐れと戦うために「新入社員に自分の経験を伝える」という行動があるのか、それとも「相手を自分より上に行かせたい」という無意識的な願いがあるのか——その『葛藤』がキャラクターを立体化させます。


## 4. 「『舞台上での身体の使い方』を『その人物の人生観』から導出する」

演技は「言葉」だけではなく「身体」です。AIに「この上司の『47年の人生』が『舞台上の動き』にどう現れるべきか』を『歩き方』『立ち方』『視線の向き』『手の使い方』で描写してください。その上司の『疲労感』『意地』『さりげない優しさ』『やり場のない怒り』が『同時に存在』するような身体表現」を作成してもらいます。完璧な『直立』ではなく「わずかに右肩が下がっている」という『人生の重さ』が無意識に現れる身体——それが『本物の役作り』になります。


## 体験談: 「『定年を控えた上司』というキャラが『初老の人間』に変わった2週間」

私は長年、舞台上で人物を演じることに携わってきました。しかし最近、ある役の「定年を控えた上司」という設定に、どうにも表面的な理解しか及ばず、深みに欠けると感じていました。そんな時、このAIアプリを知り、藁にもすがる思いで試したのです。

アプリは「この人物が子供の頃に抱いた夢は？」「キャリアの中で最も後悔したことは？」「家族との関係で何が一番の支えになったか？」など、台詞の裏にある人生全体を問うてきました。AIとの対話を通じて、私は単なる「定年を控えた上司」としてではなく、夢破れ、家族との葛藤を抱え、老いと向き合う「一人の初老の人間」として彼を捉え直すことができました。

二週間後、私の役作りは劇的に変わっていました。舞台上の「瞬間の選択」が、彼の過去の経験や内的な葛藤から導き出されるようになり、演技に圧倒的な説得力が増したのです。まさに、表面的な設定が、血の通った生身の人間に生まれ変わった感覚でした。このアプリは、私の役者人生に新たな扉を開いてくれました。
