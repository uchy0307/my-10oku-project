# 【AI対話アプリ】200の問い #114「役作り・キャラクター分析」

このアプリは「役作り・キャラクター分析」のためのAI対話プロンプト集です。演じる役を『表面の台詞』ではなく『その人物の人生全体』から構築し、舞台上の「瞬間の選択」を『キャラクターの内的論理』から導き出します。

---


## 1. 「その人物の『20年前』『今』『20年後』を3点で定義する」


## 2. 「『台本の場面』で、その人物が『本当に言いたいこと』を見つける」

台詞は「セリフの表面」と「本当の感情」が異なります。AIに「舞台のこのシーンで課長が『新入社員に教える』という台詞がありますが『本当のところ、その課長は何を伝えたいのか』『その台詞の『下層』に何があるのか』を『その課長の内的独白』として書いてください」と頼みます。「がんばれ」という台詞の下層には「俺みたいになるな」という無意識の願いがあるかもしれません。その『本当の感情』が舞台上の一言一言の『質感』を変えます。


## 3. 「『その人物が恐れていること』『その人物が望んでいること』を二者択一で言語化する」

キャラクターは「矛盾」で生きています。AIに「この課長が『心の奥底で恐れていることは何ですか』『その恐れから逃げるために『何を成し遂げたいと思っていますか』を『その課長の目線』で『今この瞬間』に書いてください」と頼みます。恐れは「人生の失敗」「存在の空虚感」「定年後の無用感」。その恐れと戦うために「新入社員に自分の経験を伝える」という行動があるのか、それとも「相手を自分より上に行かせたい」という無意識的な願いがあるのか——その『葛藤』がキャラクターを立体化させます。


## 4. 「『舞台上での身体の使い方』を『その人物の人生観』から導出する」

演技は「言葉」だけではなく「身体」です。AIに「この課長の『47年の人生』が『舞台上の動き』にどう現れるべきか』を『歩き方』『立ち方』『視線の向き』『手の使い方』で描写してください。その課長の『疲労感』『意地』『さりげない優しさ』『やり場のない怒り』が『同時に存在』するような身体表現」を作成してもらいます。完璧な『直立』ではなく「わずかに右肩が下がっている」という『人生の重さ』が無意識に現れる身体——それが『本物の役作り』になります。
