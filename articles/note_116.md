# 【AI対話アプリ】200の問い #116「YouTube企画・フック作成」

このアプリは「YouTube企画・フック作成」のためのAI対話プロンプト集です。「面白い企画を思いついた」から「48時間で視聴者を引き込む冒頭10秒」「クリック率を引き上げるサムネイル」「共有されるタイトル」までを、心理学と構成論から言語化します。

---


## 1. 「視聴者の『見る理由』を『具体的なペイン』で定義する」

動画の企画を考える際、私たちはつい「何を話したいか」に目が行きがちです。しかし、本当に大切なのは「視聴者がなぜその動画を見るのか」という視点。このセクションでは、その『見る理由』を、単なる興味ではなく、視聴者が抱える『具体的な悩みや課題（ペイン）』として深く掘り下げて言語化します。以前の私は「多くの人が楽しめる動画を作ろう」と漠然と考えていましたが、それでは誰にも刺さらない企画になりがちでした。このアプリで「視聴者のどんな不満や不安を解消できるのか？」と問い続けた結果、企画の核が明確になり、フックが驚くほど鋭くなるのを実感しました。

例えば、視聴者が「忙しい日々のストレスを解消したい」という漠然とした願望を持っている場合、その裏には「肩こりがひどい」「寝つきが悪い」「集中力が続かない」といった具体的なペインが隠れているかもしれません。これらのペインを明確にすることで、あなたの動画は、悩みを抱える視聴者にとって「まさにこれだ！」と感じさせる解決策となり得ます。

AIへの問いかけ例としては、「私の企画『初心者向けプログラミング学習』について、視聴者が抱える具体的なペインを3つ教えてください」や「これらのペインを持つ視聴者は、動画の冒頭でどんなメッセージを受け取れば、最も強く引き込まれるでしょうか？」などが考えられます。このアプローチで、視聴者の心に深く響く企画を生み出しましょう。


## 2. 「『冒頭10秒のフック』を『ショック』『好奇心』『感動』の3パターンで作り分ける」

冒頭10秒は、視聴者が動画を見続けるか離脱するかを決める決定的な時間です。このアプリでは、心理学と構成論に基づき、「ショック」「好奇心」「感動」の３つのフックで視聴者の心を瞬時に掴む方法を言語化します。

「ショック」は、予想外の事実や常識を覆す映像で視聴者を驚かせ、思考を停止させます。「好奇心」は、「なぜ？」という疑問を投げかけ、続きを見ずにはいられない心理を生み出します。そして「感動」は、共感や感情移入を促し、視聴者の心を揺さぶり、動画への没入感を深めます。

以前は漠然と「面白い冒頭」を考えていた私ですが、この３パターンで意図的に作り分けるようになってから、動画の視聴維持率が格段に向上したことに気づきました。狙った感情を視聴者に与える設計が可能になったのです。

例えば、「この企画で常識を覆す冒頭10秒のセリフと映像アイデアを『ショック』型で提案してください」や、「視聴者が思わず『え、どういうこと？』と前のめりになるような『好奇心』型の冒頭フックを複数教えてください」といった形でAIに問いかけることで、具体的なアイデアを瞬時に引き出せます。視聴者の感情に直接訴えかける冒頭フックを、ぜひこのアプリで体験してください。


## 3. 「『サムネイル設計』を『色×顔×テキスト』の三角形で考える」

サムネイルは「アート」ではなく「科学」です。AIに「YouTubeで『高クリック率』のサムネイルを観察して『共通する要素』を抽出してください。『背景色が暖色系』『顔が大きく映っている』『テキストが『黒い背景に黄色い文字』『驚きの表情』など」と聞きます。その『共通パターン』を自分の動画に適用すると「一般的なサムネイル」から「つい押してしまうサムネイル」に変わります。


## 4. 「『タイトル』を『感情語+数字+ターゲット』で構成する」

クリック率の40%は「タイトル」で決まります。AIに「『高クリック率のYouTubeタイトル』の『パターン』を5個抽出してください。『3つの』『7つの』『衝撃』『驚き』『絶望』『逆転』——『感情語+数字+ターゲット』の『テンプレート』。あなたの『人生設計の動画』で『その『テンプレート』を当てはめたら、どんなタイトルになるか』を10案作成してください」と頼みます。


## 体験談: 「企画のアイデア」が「48時間で『3000再生の動画』に変わった」

長年温めていた企画のアイデアはあったものの、いざ動画にしようとすると、どうすれば人に見てもらえるのか全く分かりませんでした。漠然とした不安を抱えながら、この「YouTube企画・フック作成」アプリを試したのです。

私が企画を伝えると、AIはすぐに「冒頭10秒で視聴者を惹きつける構成」を具体的に提示してきました。心理学に基づいた言葉選びや、瞬時に興味を引く映像の切り口まで、まるで隣にプロのディレクターがいるかのようでした。そのアドバイスに従って動画を作り、サムネイルとタイトルもAIが提案したクリック率を意識したものに修正。すると、公開からわずか48時間で、動画はまさかの3000回再生を突破したのです。

今まで自己満足に終わっていた企画が、アプリとの対話を通じて、視聴者に届くコンテンツへと見事に昇華しました。この体験は、私の動画作りに対する意識を根本から変えてくれました。
