# 【AI対話アプリ】200の問い #126「教師・生徒の心を開く問い」

このアプリは「教師・生徒の心を開く問い」のためのAI対話プロンプト集です。部下・子ども・生徒など「あなたの指導下にある人」の心理的な抵抗や黙秘を、一つの問いで解きほぐすAI提問ツールです。

---


## 1. 「正解を求める質問」ではなく「本音を許可する質問」に翻訳する

上司が「なぜ目標達成できなかったのか」と聞くと、部下は「正解を探して」「失点しない回答」を用意します。本音は隠れたままです。AIに「部下が心理的に『本当の困難を言ってもいい』と感じる質問に翻訳してください。現在の質問『なぜ目標達成できなかったのか』の代わりに」と聞きます。例えば「目標達成に向けて、何が一番困っていた?」「もし一つだけ、上司に伝えたいことがあるなら」といった問い方に変わる。正解を求めない質問は、本音を許可する質問です。


## 2. 「部下の立場から見たあなた」を認識する

多くのマネージャーは「相手がなぜ心を開かないか」の原因を、相手に帰属させます。しかし実は「部下から見た上司」の姿が、相手の心理的な障壁を作っている可能性があります。AIに「あなたが部下の立場だったら、あなたを『上司として』どう見ていますか。『怖い』『厳しい』『応援してくれない』など、率直に」と聞きます。その率直さが、部下が心を開かない理由を照らします。


## 3. 「失敗を許可する言葉」を事前に用意する

部下が本音を話す時、多くの場合「失敗や弱さの告白」が含まれます。「実は営業の電話が苦手で」「クライアント対応で泣いてしまい」といった弱さを言える環境を作ることが先決です。AIに「部下が『失敗や弱さを言ってもいい』と感じるために、あなたが事前に言うべき一言は」と聞きます。「目標達成できなかった時点で、その理由は何でもいい。大事なのは、今後どう変えるかだ」といった許可の言葉が、部下の心を開きます。


## 4. 「沈黙の時間」を味方にする

対話の中で相手が沈黙すると、多くの人は「困った」と感じて次の質問を投げてしまいます。しかし沈黙こそが「相手が本当に考えている時間」です。AIに「相手が沈黙した時、上司として何をすべきか。無言で待つことで何が起きるか」と聞きます。沈黙の中で、相手は自分の本当の気持ちと向き合い始めるのです。


## 体験談: 「正解質問を1つの許可質問に変えて、部下の真の課題が見えた」

正直に書きます。最初の面談では「なぜ目標達成できなかったのか」と聞き、田中君は「営業努力が足りませんでした」と返しました。その正解回答では何も見えません。1週間後、AIの助言を受けて、質問を変えました。「目標達成に向けて、何が一番困っていた?」。その問いに対して、田中君は1分間沈黙しました。その沈黙を、私は話しかけずに待ちました。その後、彼は涙ぐみながら「正直、営業電話をするのが、すごく怖くて」と言いました。元々営業職志望ではなく、親の勧めで営業部に配置された経緯があったのです。その本音が見えた時点で「課題」が全く異なるものになりました。営業目標の未達ではなく「職種適性」が本当の課題だったのです。その後、彼は営業企画職への異動を希望し、現在はその職で顕著な成果を出しています。「正解質問」では絶対に見えなかった課題を、「本音許可質問」が照らしたのです。

## 🎯 上級プロンプト: 「部下の黙秘の根拠」推定法

面談で心を開かれないと感じたら、AIに次のように頼みます:
「この部下が『本当のことを言わない』理由を、相手の心理的立場から推定してください。相手が『こう見られるのが怖い』『こう評価されるのが嫌だ』という恐怖の背景まで、200字で。」

```
あなたは、指導を受ける側の心を開き、本音を引き出すことに特化した対話コーチです。相手の心理的抵抗を和らげ、信頼関係を築くための問いかけを段階的に構築し、最も効果的な方法を提案します。

まず、以下の情報について順に教えてください。
1.  どのような状況で、誰に対して心を開く問いかけをしたいのか具体的に教えてください。（例：高校生のA君が最近無気力で、私と目を合わせません。）
2.  その相手は普段どのような性格で、何に興味を持っていそうでしょうか？また、最近何か変化がありましたか？（例：元々は明るくリーダーシップを発揮するタイプでしたが、最近はSNSばかり見ています。）
3.  あなたはその問いかけを通じて、最終的に相手にどうなってほしいと願っていますか？（例：なぜ無気力なのか本音を聞き出し、前向きな気持ちを取り戻してほしいです。）
4.  これまでの対話で、相手が心を開かなかった経験があれば、その時の状況と、あなたが投げかけた言葉も教えてください。（例：『どうしたの？何かあった？』と聞いても『別に』としか言いませんでした。）
5.  私が提案する問いかけは、どのようなニュアンスを重視すべきだと思いますか？（例：直接的な問いかけではなく、間接的に心境を探るようなものがいいです。）

これらの情報をもとに、相手の心を開くための具体的な問いかけを一つ提案してください。そして、その問いかけを実践する上での具体的なアドバイスを3つ提示してください。
```

この問いにより、相手の沈黙が「単なる反抗」ではなく「合理的な自己防衛」であることが見える。その理解が、マネージャーの次の一言を、格段に適切にします。


## 📖 追加体験談・失敗回避リスト
