# 【AI対話アプリ】200の問い #132「占い師・ポジティブ整形」

このアプリは「占い師・ポジティブ整形」のためのAI対話プロンプト集です。人生の悲観的な解釈を「別の視座」で読み替え、同じ出来事から「希望」を引き出す思考法を学ぶAI対話ツールです。

---


## 1. 「制限時間」を「余裕」に言い換える

占い結果が「40歳までが結婚ターニングポイント」という場合、一般的には「時間がない」という絶望を与えます。しかし逆視点では「40歳までまだ時間がある」という希望です。AIに「この占い結果『◯◯がターニングポイント』を『逆に読むと、あなたに何という猶予があるのか』と表現し直してください」と聞きます。同じ事実を、時間の「制限」ではなく「猶予」として読み直すことで、心理的な圧迫感が消える。


## 2. 「弱み」を「伸びしろ」に読み替える

占い結果は「対人運が弱い」「金運が弱い」という弱点を示すことがあります。それを「伸ばしの余地がある」「改善の可能性がある」と読み替える。AIに「この占い結果『◯◯が弱い』を『◯◯の可能性が秘められている』と言い直したら、何が見えますか」と聞きます。同じ弱点が「改善の入口」に変わる。


## 3. 「課題」を「使命」に読み替える

占い結果は「苦労がある」「試練が待つ」という課題を示すことがあります。それを「その課題を乗り越えた時に得られる成長」として読み替える。AIに「この占い結果『◯◯という試練』を『その試練を通じて、あなたが手に入れるもの』として500字で物語化してください」と聞きます。同じ試練が「成長への道」に変わる。


## 4. 「運命」を「選択肢」に読み替える

占い結果は「決定的」に聞こえます。「この人は一生独身」「この人は貧乏」。しかし実は「そういう傾向がある」という確率的な予測でしかありません。AIに「この占い結果『◯◯である』を『◯◯になりやすい。だがあなたの選択で変わる』と言い直したら」と聞きます。同じ結果が「改変可能な展望」に変わる。


## 体験談: 「絶望的な占い結果を『希望の読み』に変えて、クライアントの人生が動いた」

正直に書きます。その30代女性に「結婚のターニングポイントは40歳」と伝えた時、彼女は涙ぐんでいました。セッション後、私は自分の職業に疑問を感じていました。「占いは人を絶望させるのか」と。その夜、その占い結果を「逆に読む」練習をAIとしました。「40歳までがターニングポイント」→「今、あなたは40歳までの時間を自由に使える状態にある。その時間で『最高の人生パートナーを選ぶ準備』ができる」という読み替え。「対人運が弱い」→「対人スキルを高める余地が最も大きい。あなたはこれから『最高の対人スキル』を身につけられる人なのです」という読み替え。翌週、その女性を再度セッションに招き「逆読み」を伝えました。その時、初めて彼女の眼差しが「絶望」から「希望」に変わりました。1年後「占い師さんのおかげで、素敵な人と出会いました」という報告を受けました。同じ占い結果が「人を絶望させるか」「人を希望させるか」は「読み方」次第だったのです。
