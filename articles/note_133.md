# 【AI対話アプリ】200の問い #133「Vチューバー・キャラ設定」

このアプリは「Vチューバー・キャラ設定」のためのAI対話プロンプト集です。あなたの理想のキャラクターを「行動パターン」「会話スタイル」「世界観」「深い設定」で立体化させるAIキャラクター構築ツールです。

---


## 1. 「第一印象」から「隠された深さ」へ段階的に構築する

キャラクターの印象は「初見では見えない深さ」を持つことで、ファンから愛されます。AIに「このキャラクターの『第一印象（明るい、ポップ、など）』『1ヶ月見た時の『ちょっと見える複雑さ』『3ヶ月見た時の『実は抱えている悩み』を段階的に構築してください」と聞きます。深さがあるキャラは「応援したくなる」ファン心理を生みます。


## 2. 「会話パターン」を5-6個の定型句で定義する

Vチューバーは配信を継続する中で「このキャラらしい言い方」を作る必要があります。AIに「このキャラクターが頻繁に使う『定型句』『口癖』『リアクション』を5-6個設計してください。その定型句が『このキャラの人格』を表すもの」と聞きます。その定型句が「キャラの記号」になり、ファンが真似するようになる。


## 3. 「何が好きで、何が嫌いか」の嗜好を具体化する

キャラクターの説得力は「嗜好の一貫性」から生まれます。AIに「このキャラクターが『好きな食べ物』『好きなゲームジャンル』『好きな動物』『実は嫌いなこと』を設定してください。その嗜好から『このキャラは実はどういう心理の持ち主か』が見える設定で」と聞きます。嗜好が「キャラの心理」を表現する媒介になります。


## 4. 「配信中に起きるトラブル時の反応パターン」を用意する

実際の配信では「ゲームが落ちる」「音声が乱れる」「予期しないコメントが来る」という不測の事態が起きます。AIに「このキャラが『ゲーム画面が固まった時』『視聴者から悪意あるコメントをされた時』『自分が言い間違えた時』に、どう反応するか、3つのシナリオを200字ずつで」と聞きます。その反応パターンが「キャラクターの揺らぎない軸」を表現します。


## 体験談: 「曖昧なキャラを『行動パターンと会話スタイル』で立体化させて、初配信で3000人のファンを掴んだ」

正直に書きます。最初、私は「可愛いアバターなら、配信の中で自然とキャラが出てくる」と甘く考えていました。ただし、配信本番の1ヶ月前にAIに「キャラクター構築」をさせました。第一印象は「元気で明るい」でしたが、隠された深さとして「実は新人で緊張しやすい」という設定を加えました。口癖は「わわっ」「びっくりしたぁ」など、ビックリした時のリアクションを5個設定しました。嗜好として「実は怖いホラーゲームが好きだが、怖がりなので配信中は悲鳴を上げる」という矛盾を設定しました。それらが配信中に「このキャラらしい瞬間」を作りました。初配信でゲーム画面が乱れた時「わわわ!技術的な問題ですね。ごめんなさい!」というリアクションが「このキャラは責任感強い性格」という印象を与えました。初配信の3時間で3000人のフォローを獲得しました。キャラクターの「立体性」が、視聴者の「応援したい気持ち」を生んだのです。
