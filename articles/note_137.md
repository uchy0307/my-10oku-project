# 【AI対話アプリ】200の問い #137「美容師・感動カウンセリング」

このアプリは「美容師・感動カウンセリング」のためのAI対話プロンプト集です。顧客の「髪型の好み」ではなく「理想の自分」を引き出し、その実現の喜びを一緒に作る美容師のカウンセリングスキルを学ぶツールです。

---


## 1. 「髪型の希望」ではなく「人生の願い」を聞く質問に変える

カウンセリングで「どんな髪型が好きですか」と聞くと、顧客は「雑誌のこの人みたいな」と答えます。でも本当に聞くべきは「あなたは、今後、どんな自分になりたいのか」です。AIに「『髪型について』という表面的な質問の奥に隠れた『この顧客の人生における願い』を引き出す質問は何か」と聞きます。例えば「最近、自分の見た目について『変わりたい』という想いはありますか?」という問いが、より深い層の答えを引き出します。


## 2. 「変わることへの不安」を認識する

顧客が髪型を変えたい時、その背景には「不安」があることが多い。新しい髪型で「浮かないか」「似合わないか」「周囲の反応は」という恐れです。AIに「顧客が『髪を大きく変えたい』と言う時、その奥に隠れた『変わることへの不安』は何か。その不安を認識し、安心させるカウンセリングは」と聞きます。その認識が「安心のカウンセリング」を作ります。


## 3. 「施術中の心理変化」を見取る

施術中、顧客の表情や言葉は「セルフイメージの変化」を示しています。AIに「カット中に『前髪がこんなに短くていいのかな』と不安そうなつぶやき。その瞬間、美容師は何を言うべきか。顧客の不安を安心に変える一言は」と聞きます。その一言が「この美容師は信頼できる」という感情を作ります。


## 4. 「変わった後の『新しい自分』を確認させる」

施術完了後「鏡を見て」と鏡を示すだけではなく、顧客の「新しい自分への発見」をサポートすることが大事です。AIに「施術完了後、顧客に『新しい自分』を発見させる質問は何か。『いかがですか』だけではなく、『この新しい自分で、今後やってみたいことは何ですか』といった問いで」と聞きます。その問いが「新しい髪型への納得」を作ります。


## 体験談: 「『髪型の希望』を聞く代わりに『理想の自分』を聞いたら、顧客が涙ぐんだ」

正直に書きます。最初、私は「どんな髪型が好きですか」というテンプレート質問をしていました。その40代女性も「傷んだ髪をまとめたい」と表面的に答えていました。その時、AIに「この顧客の本当の願い」を考えさせました。「傷んでいる髪をまとめたい」という言葉の背景には「最近、自分の見た目に自信がない。でも変わりたい気もある」という複雑な感情があるのではないか。そこで質問を変えました。「最近、何か『変わりたい』という想いはありますか?」その問いに、彼女は「実は、今年で50歳。20年以上同じ髪型です。何か新しく始めたいという気持ちがあって」と答えました。その本音が聞けた瞬間、カウンセリングの内容が変わりました。単純に「髪をまとめる」のではなく「50歳、新しい人生への入口として『顔周りを生かした髪型』」という提案に変えました。施術中「本当に短くしていいですか」という不安を聞いた時「この短さが、あなたの顔の輪郭をきれいに見せます。新しい自分が見えてきます」と言いました。完成後、鏡を見た時、彼女は涙ぐみました。「自分がこんなに変わるなんて。50歳から新しく始められるんですね」。その涙が「美容施術の本質」を教えてくれました。髪型を変えることは「人生を変えること」だったのです。
