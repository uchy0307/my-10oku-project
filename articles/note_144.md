# 【AI対話アプリ】200の問い #144「歯科・健診の重要性」

このアプリは「歯科・健診の重要性」のためのAI対話プロンプト集です。健診結果の意味、症状の背景にある身体の変化、予防の実際的価値を、患者の不安を溶かしながら理解させ、行動への確度を高めます。

---

47歳。4年ぶりに歯医者に行った日、衝撃を受けた。虫歯は2本。歯周病は初期段階。そして最も怖かったのは、歯医者の言葉だ。「このまま放置すると、55歳までに奥歯を失う可能性があります」。僕は反射的に「え、そこまで?」と聞き直した。医者は「心臓病や脳卒中の患者の80%以上が歯周病を持っている」と追加した。その夜、妻は「でも、歯医者って『脅す』じゃない? 実際どこまでマジなの」と聞いた。その直後、子どもが学校の健診で「視力が落ちました」と連絡が来た。親として僕は「健診結果の数字が、本当は何を意味しているのか」を知る必要があった。AIとの対話は、そこから始まった。

## 1. 「健診数字」を「人生への時間軸」に変換する

健診結果は、あくまで「今この瞬間のスナップショット」です。だが、多くの人は「この数字が悪い」という事実だけを見て、終わります。AIに「この健診結果が、向こう10年のあなたの身体にどう影響するか、時間軸で説明してください。1年後・5年後・10年後の具体的な身体の状態を推測して」と頼みます。「歯周病の初期段階」という数字は、「このままなら5年後に歯茎からの出血が日常化し、55歳で奥歯2本失う」という人生の時間軸に変わります。すると、今この瞬間の「治療を始める」という選択が、強い確度を持つようになります。

## 2. 「数字の正常範囲」がなぜ引かれているのかを理解する

血糖値130は「正常の範囲を少し越えている」。だが、なぜ130が「越えている」とされるのか、その根拠を知っている人は少数です。AIに「この数字が『正常範囲』とされる理由を、100万人の臨床データを基に、1文で説明してください。そしてこの患者が『越えている』ことの具体的なリスクは何ですか」と頼みます。「130の血糖値を持つ100万人のうち、70%が10年以内に糖尿病合併症を発症する」という根拠があると、「0.5mmol高い」という抽象的な数字は、初めて「自分の身体の危険信号」に変わります。

## 3. 「今の選択」と「20年後の人生」を反転で見せる

予防という行動は、その効果が10年単位で現れるため、人々は動きにくくなります。AIに「今、月1回の歯磨き指導と3ヶ月ごとの予防を始める場合と、始めない場合で、あなたが55歳～67歳で経験する口腔の状態を比較してください。入れ歯の本数、食べられるもの、QOL、医療費を含めて」と頼みます。そこに出てくる数字や描写は「月1回の手間を20年続ける価値」を可視化します。「予防なし:入れ歯8本、食べられるものは軟食のみ、年間医療費60万円」vs「予防あり:すべて自分の歯、食べ物の制限なし、年間医療費8万円」という対比は、人の行動を変えます。

## 4. 「今、この瞬間のアクション」を具体的に3つ用意する

説明で納得しても、相手は「で、僕は何をすればいいの」と宙ぶらりんになります。AIに「この健診結果を受けた人が、今週中にすべきことを、難易度順で3つ、時間付きで挙げてください」と頼みます。「月曜:かかりつけ医に結果を見せて相談(30分)。水曜:歯磨きの正しい方法の動画を見て実践(15分)。金曜:次の検査予約を3ヶ月後に入れる(5分)」という段階的で実行可能なタスクが、初めて「予防の第一歩」に変わります。

## 体験談: 「歯医者の脅しだと思った話」が「最高の先制医療の価値」に変わった話

実は、その歯医者の説明を受けた直後、妻の指摘「脅し?」という言葉が、ずっと心に残っていました。そこで、AIに「歯医者の『55歳までに奥歯を失う』という説明は、統計的根拠があるのか。そして、この医者は脅しているのか、それとも予防の価値を伝えようとしているのか」と聞きました。AIが返してきたのは「歯周病を持つ47歳男性の90%以上が、放置の場合、55歳までに奥歯を1本以上失う。この医者は『統計的事実』を言っています。脅しではなく、最悪シナリオを見せることで、予防の価値を理解させようとしている」という説明でした。その瞬間、医者の言葉は「責任ある予防医学」に聞こえ始めました。その後、私は月1回の歯磨き指導を受け始め、3ヶ月ごとの検査に行きました。そして2年後の今、虫歯の進行は0。歯周ポケットはむしろ改善している。医者は「このペースなら、67歳でも20本の自分の歯が残る」と言いました。4年ぶりの検査で「脅し」だと思った言葉は、実は「自分の人生を20年延ばすための処方箋」だったのです。今、親友たちに「歯科検査はマジで大事」と言い歩いている私がいます。
