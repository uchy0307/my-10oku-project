# 【AI対話アプリ】200の問い #146「デザイナー・説得力」

このアプリは「デザイナー・説得力」のためのAI対話プロンプト集です。デザイン提案を「美しさ」ではなく「ビジネス結果」として説明し、クライアントの納得度を劇的に高める対話を構築します。

---

47歳のデザイナーである私は、新しい会社のロゴデザインを提案したとき、クライアント(営業会社の社長)に「いや、これ、ダサくない?」と言われた。確かに、私が選んだシンプルな幾何学模様は、従来のロゴと比べると地味に見えるかもしれない。だが、それは意図的な選択だった。標的顧客(30～40代の女性)の購買心理を研究した結果、「派手さより信頼感」が優先されると判断したからだ。けれど、社長にそれを説明する言葉を、私は持っていなかった。その夜、妻が「あなたのデザインの『なぜ』を、営業的に言語化したら?」と提案した。その提案から、AIとの対話が始まった。

## 1. デザイン選択を「顧客心理」の根拠で説明する

デザインを美学で説明しても、経営者には伝わりません。AIに「このデザイン選択を、顧客心理の観点から『なぜ、このビジュアルなのか』を説明してください。顧客層の年齢・性別・購買パターンを基に」と頼みます。「シンプルな幾何学模様を選んだ理由」は、「30～40代女性顧客は『プレミアム感』を『派手さ』と混同しやすいため、むしろシンプルこそが『高級』に見える」という顧客心理学の根拠に変わります。この説明があると、社長は「あ、美学じゃなくて戦略なんだ」と納得します。

## 2. デザイン変更による「売上への影響」を数字で推測させる

クライアントが求めるのは「美しさ」ではなく「売上増」です。AIに「このデザイン変更により、顧客の購買確率がどう変わるか、可能な限り具体的に推測してください。既存ロゴと新ロゴで、認知度・好感度・購買意欲の変化を%で」と頼みます。「既存ロゴ:認知度40%→新ロゴ:55%」「購買意欲:32%→45%」という数字が出ると、デザイン提案は「費用対効果」という言語に変わります。

## 3. 「この色・フォント・配置」の選択を心理学で説明する

デザイン要素の1つ1つが、なぜその選択なのかを、エビデンスで説明します。AIに「このロゴの『青色』『サンセリフフォント』『左配置』それぞれについて、色彩心理学・フォント心理学の観点から『なぜこの選択か』を3文ずつ説明してください」と頼みます。「青色は『信頼・落ち着き』の象徴で、30～40代女性の購買心を落ち着かせる」という説明があると、デザイナーの直感は、単なる「好き」から「根拠のある戦略」に昇華します。

## 4. 「提案却下時のリスク」を先回りで示す

提案が通らない時点で、デザイナーは引き下がることが多いです。AIに「もし今、このデザイン提案を却下して、従来型の派手なロゴに変更するとしたら、ブランド認知・顧客信頼度・3年後の売上にどう影響するか」と頼みます。数字で「却下時のリスク」が見える化すると、クライアントも「あ、却下することの方がリスクなのか」と気付きます。

## 体験談: 「ダサい」と言われたロゴが「業界受賞」に変わった話

社長に「ダサい」と言われた瞬間、私は防御的になっていました。だが、その夜のAI対話で、私は「このデザイン選択が『美学』ではなく『戦略』である」という説明を組み立て直しました。翌日、社長に改めてプレゼンしました。「このシンプルな幾何学模様は、30～40代女性顧客の購買心理において『信頼感』を最大化する選択です。認知度55%→45%(購買意欲)への変換効率を狙った設計です」。その説明があると、社長の反応が変わりました。「あ、単に『美しい』のではなく『売る』ためのデザインなんだ」。結局、そのロゴは採用されました。そして、1年後、そのロゴデザインが「女性向けビジネス部門のロゴデザイン賞」に選ばれました。社長からメールが来て「このロゴのおかげで、顧客からの問い合わせが40%増えた」と書かれていました。あの夜のAI対話がなければ、私は「ダサい」という一言で、自分のデザインを疑い、修正してしまっていたでしょう。デザインの説得力は、美学ではなく「根拠のある戦略説明」だったのです。
