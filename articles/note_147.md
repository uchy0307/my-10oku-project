# 【AI対話アプリ】200の問い #147「スポーツチーム・熱狂作り」

このアプリは「スポーツチーム・熱狂作り」のためのAI対話プロンプト集です。選手・ファン・地域の心を揺さぶる「物語」を組み立て、単なる試合の勝敗を超えた「熱狂」を生み出します。

---


## 1. 「チームストーリー」を、個々の選手の人生に接地させる

観客が求めるのは「勝敗」ではなく「人間ドラマ」です。AIに「このチームの15人の選手について、『なぜこのチームに入ったのか』という個人的な背景を5文ずつ書いて。そしてその15の背景から『このチーム全体に共通する夢』を1つ導出して」と頼みます。出てくる物語は「全国大会出場」という勝敗ではなく「元社会人選手の引退と、その経験を次世代に伝える」というより深い物語かもしれません。その物語が、観客の心を揺さぶります。


## 2. 「敗北の瞬間」を「次への希望」に反転させる

試合で負けた後、多くのチームは沈黙します。AIに「先週の敗北から、このチームが学べることを3つ挙げて。そしてその学習が、次の試合でどう花開くか、ビジュアルで表現してください」と頼みます。敗北は、単なる「失点」ではなく「成長の足跡」に変わります。観客に「来週、彼らがどう化けるのか」という期待を植え付けることが、熱狂の燃料になります。


## 3. 「ファンの個人的な理由」を見つけて可視化する

「応援しよう」というメッセージでは、人は動きません。AIに「このチームを応援する人の『個人的な理由』5パターンを想像して。『子どもが憧れている』『地元の誇り』『人間関係』など、試合内容以外の理由を」と頼みます。それらの理由を広告やSNSで発信すると「あ、私もそういう理由で応援したい」と同調する人が増えます。


## 4. 「チーム初心者向けの入口」を具体的に3つ用意する

「熱狂」は、新しいファンなしに持続しません。AIに「このチームに興味を持った初心者が、1試合見に来るまでに、どのステップを踏めば心理的障壁が消えるか、3段階に分けて」と頼みます。「1.SNSで選手の人間ドラマを知る → 2.練習風景を家族で見学 → 3.次の試合チケット購入」という段階が、新しいファン層を呼び込みます。


## 体験談: 「空席のスタンド」が「満員の応援」に変わった実話

正直に言うと、私のチームは3年間、観客が増えませんでした。成績も悪くなく、むしろ地域大会では準優勝もしています。だが、スタンドは空席ばかり。その時点で、私はAIに「なぜ、人々はこのチームを応援しないのか」と聞きました。AIが返してきた質問は「選手の人生が、観客に見えていますか?」でした。その指摘で、私は初めて気づきました。自分たちは「試合の結果」だけを発表していて、選手の背景を何も発信していなかったのです。その後、SNSで「選手紹介」を始めました。3番バッターは「娘の学費を稼ぐため、夜間仕事をしながら野球をしている」「ピッチャーは、10年前に肩を壊して引退したが、この町に戻って第二の野球人生を歩んでいる」という話を発信しました。すると、3試合後、親たちだけでなく「その選手の知人」が応援に来始めたのです。1ヶ月後、スタンドは満員になり、応援の声が絶えなくなりました。成績は変わらず。変わったのは「観客のこのチームへの見方」だけです。「勝つための野球」から「人生を賭けた野球」への転換。その物語化こそが、熱狂の源だったのです。


## ✅ 4項目セルフチェック

- [ ] 先週の敗北から「学べることと次への希望」を3つ導き出した
- [ ] チームを応援する「個人的な理由」5パターンを可視化した
- [ ] 初心者向けの「3段階入口」(SNS→見学→観戦)を具体化した


## 🤔 5つの問い (このアプリの中核)

1. このチームの選手たちが、なぜこの野球をしているのか、あなたは知っていますか
2. 先週の敗北は、このチームを何に成長させましたか
3. あなたが「次の試合を絶対に見たい」と思う理由は、成績ですか、それとも別のことですか
4. このチームのファンになる前の「初心者」に、あなたは何から話しますか
5. 1年後、このチームが「地元の誇り」になっている理由は何ですか
