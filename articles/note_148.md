# 【AI対話アプリ】200の問い #148「ラジオ投稿・ネタ作成」

このアプリは「ラジオ投稿・ネタ作成」のためのAI対話プロンプト集です。日常の「つぶやき」を「ラジオで読まれる一話完結のネタ」に昇華させ、パーソナリティとリスナーの心を掴みます。

---


## 1. 日常のネタを「一話完結の構成」に整える

日常の「つぶやき」は、散漫で長すぎることがほとんどです。AIに「次の日常エピソードを、ラジオで読まれるレベルの『起→承→転→結』の一話完結に再構成してください。300字以内で」と頼みます。「子どもが『お父さん、臭い』と言ったので、『そりゃ仕事で汗かいてるからな』と返したら、『違う、トイレ掃除のニオイが』と言われた」という日常が、「起承転結」のテンポよく構成され直します。ラジオに読まれる投稿は、「構成」で7割決まります。


## 2. 「オチ」に共感と意外性を同時に埋め込む

ラジオリスナーは、予想を裏切られることで笑います。AIに「このエピソードの『自分はこう思ったが、実は違った』という部分を強調して、最後の一行を『パーソナリティとリスナーが笑う』という状態に設定してください」と頼みます。想定される反応を逆転させることが、オチの強さになります。


## 3. 「自分を下げ、相手を上げる」トーン調整

ラジオで受ける投稿は、「自慢」ではなく「自分のダメさ加減を笑う」というトーンです。AIに「このネタの『自分の恥ずかしさ』をより強調して、その対比で相手の『意外な正しさ』を浮き立たせるように再構成してください」と頼みます。視点を「自分が愚かである」に固定することで、共感が生まれます。


## 4. 「年代共感」を狙った細部調整

「年代共感」を狙った細部調整

ラジオ投稿において、特定の世代に深く刺さる「年代共感」は、ネタを記憶に残るものへと昇華させる鍵となります。この機能では、あなたの日常の「つぶやき」を、ターゲットとする世代の心に響く細やかな視点や表現で調整します。

例えば、子育て真っ最中の親御さんが「それ、わかる！」と共感するような育児の喜びや葛藤、あるいは日々の仕事に奮闘する世代が「まさに自分たちのことだ」と感じるような、キャリアや人間関係の機微。AIは、それぞれの世代が共有する経験や価値観、流行、言葉遣いなどを巧みに織り交ぜ、より具体的な描写やリアリティのあるシチュエーションを提案します。

私自身、以前は漠然としたネタばかりでしたが、ある時「小さなお子さんのいる親御さんが共感するような表現に調整して」とAIに問いかけたり、「日々の仕事に奮闘する世代が『うんうん』と頷くような視点で、このエピソードを深掘りしてほしい」と依頼するようになってから、ネタが格段に生き生きとし始めました。自分の何気ない出来事が、特定のリスナー層に深く刺さる一話完結の物語へと変わる瞬間に、毎回驚きと感動を覚えています。狙った世代の記憶に残り、パーソナリティが思わず「わかる！」と口にするような、共感度の高いネタを一緒に作り上げましょう。


## 体験談: 「ボツ投稿3回」から「読まれた投稿」へ至った話

いつもラジオに投稿してはボツ。特に目立った出来事もない日常をどう面白く伝えるか、若い頃のような勢いも既になく、どこか諦めにも似た感情でいました。三度目のボツ通知を受け取った時、もう諦めようかとも思ったほどです。

そんな時、この「ラジオ投稿・ネタ作成」アプリを見つけました。半信半疑で、今日の帰り道にふと見た夕焼けに関する漠然とした感想を入力。「これのどこが面白くなるのか」と思っていた私に、AIは「その夕焼けを見て、あなたはどんな感情を抱きましたか？」「その感情は、どんな日常の出来事と結びつきますか？」といった具体的な問いかけを投げかけてきました。

ただの日常のつぶやきが、AIの問いかけによって見事に磨き上げられていく感覚。視点を変え、オチを考え、共感を呼ぶ言葉を選ぶ手助けを受け、一話完結のネタとして成立する形へと導かれました。

そして、投稿。数週間後、聴き慣れたパーソナリティの声で、私のネタが読まれたのです。まさか、と耳を疑いました。自分の何気ない日常が、誰かの笑い声に変わる。あの時の喜びは、何物にも代えがたいものでした。もうボツを恐れません。

## 🎯 上級プロンプト: 「パーソナリティの笑い方」シミュレート法

ネタが完成したら、AIに次のように頼みます:
「このネタを読んだパーソナリティが、どこで笑い、どうコメントすると思いますか? その台詞を実際に台本のように書いてください。」

```
あなたはラジオ投稿ネタ作成の専門家であり、日常の出来事をパーソナリティとリスナーの心に響く一話完結のネタに昇華させる名コーチです。あなたの力を借りて、面白いラジオネタを作成しましょう。

それでは、まず最初の質問です。
あなたがラジオ投稿のネタにしたいと考えている、最近あった出来事やつぶやきを具体的に教えてください。些細なことでも構いません。

続いての質問です。
その出来事の中で、特にあなたが「面白い」「共感できる」「驚いた」と感じたポイントは何ですか？パーソナリティが食いつきそうな、核となる部分を深掘りしてみましょう。

さらに、そのポイントを最大限に活かし、リスナーが情景を思い浮かべやすいような具体的な描写や、クスッと笑えるようなオチを付けるとしたら、どのような表現が考えられますか？一話完結のストーリーとして組み立ててみましょう。

上記を踏まえて、あなたが提供した情報から最高のラジオ投稿ネタを一緒に作り上げましょう。

最後に、具体的な次の一歩として、以下の3つの提案をします。
提案1: 作成したネタを声に出して読んでみてください。実際に耳で聞くことで、リズムやテンポ、言葉の響きが分かります。
提案2: 投稿したいラジオ番組の過去の読まれたネタや、パーソナリティの話し方を参考に、さらにネタを磨き上げましょう。
提案3: 今回のプロセスを参考に、他の日常のつぶやきからも新たなラジオネタを発掘してみてください。発想の転換が習慣になります。
```

その台本を読んで「自分もそこで笑う」なら、そのネタは確度が高いです。
