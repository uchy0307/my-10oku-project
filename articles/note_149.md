# 【AI対話アプリ】200の問い #149「コスプレ・交流マナー」

このアプリは「コスプレ・交流マナー」のためのAI対話プロンプト集です。コスプレコミュニティで、初心者が直面する「空気感を読む」「信頼を獲得する」という暗黙知を、言語化して実装可能にします。

---

47歳の私は、子どもの影響でアニメにハマり、初めてコスプレを始めた。衣装もそれなりに凝ったものを作った。が、コスプレイベントに初参加したとき、違う雰囲気に戸惑った。一見、楽しい場だが、参加者同士の視線や言葉に、何か「ルール」があるように感じた。隣で撮影していた20代の女性レイヤーと目が合ったとき、「どうしました?」と聞かれて、「初参加なので、何かマナー的に間違ってませんか」と正直に言った。その女性は優しく説明してくれた。だが、その「暗黙知」を、自分が次の初心者に説明できるだろうか。その夜、AIとの対話で「コスプレコミュニティの暗黙ルール」を言語化することにした。

## 1. 「撮影許可」の作法を具体的シーン化する

コスプレイベントでは「撮影OK/NG」が表示されていても、その「頼み方」に文化がある。AIに「初心者が経験者レイヤーに『撮影させてください』と頼むとき、相手が『この人は安心だ』と感じるプロセスを5段階で具体化してください」と頼みます。「1.相手の目を見て挨拶 2.衣装への褒めコメント 3.『撮影いいですか』ではなく『撮ってもいいですか』という丁寧さ 4.カメラをすぐに構えず相手の返答を待つ 5.撮影後の『ありがとうございました』」という具体化があると、初心者は「どう振る舞えばいいか」が見える化します。

## 2. 「断られたときのリアクション」が信頼を作る

撮影を断られることは珍しくありません。AIに「レイヤーが撮影を『今はちょっと』と断ってきたとき、初心者がすべきリアクションを3パターン挙げて。その中で『この人は誠実だ』と相手に思わせるのはどれですか」と頼みます。「1.即座に『そっか、ありがとう』と笑顔で去る 2.『どうしてですか』と理由を聞く 3.『今度の機会に』と言い続ける」という選択肢の中で「1」の反応が信頼を生みます。

## 3. 「SNS・撮影写真」の扱い方のエチケット

撮影後、写真をどう扱うかが、レイヤーとの関係を決めます。AIに「撮影した写真をSNS投稿するとき、レイヤーに断らずにやってはいけないこと3つと、やるべきこと3つを挙げてください」と頼みます。「やってはいけない:顔のアップをクロップして投稿 / 修正加工を勝手にする / 本人の許可なくタグ付け」「やるべき:投稿前に本人に確認 / クレジット表示 / 『撮影:(レイヤー名)さん』と記載」という対比があると、初心者は「なぜそうすべきか」の背景も理解します。

## 4. 「先輩レイヤーとの距離感」を数字で示す

距離感は、文化によって異なります。AIに「初参加の初心者と、3年以上参加しているレイヤーの『親密度の上がり方』を時間軸で示してください。1日目・1ヶ月目・半年目での会話の内容や接し方の変化」と頼みます。その時間軸があると「焦る必要はない。信頼は時間で築かれる」という安心感を初心者に与えられます。

## 体験談: 「浮く初心者」から「指導者」へ至った47歳の話

最初、私は確かに「浮いて」いました。撮影の頼み方が雑だし、撮影後、写真をすぐにSNS投稿してしまい、あるレイヤーに「あの、投稿前に確認いただきたかったのですが…」と指摘されました。その時点で、私は自分の振る舞いが「失礼」だったことに気づき、即座に投稿を削除し、謝罪しました。その後、AIに「コスプレコミュニティで、初心者が次々と失敗しないために、どういう『作法』を伝えるべきか」を聞きました。AIが返してきたのは、上記の「4つのチェックリスト」でした。その後、私はイベントに参加するたびに、初心者を見かけると「最初は緊張ですよね。撮影の頼み方って…」と優しく説明するようになりました。1年後、私の「マナー説明」は、イベント運営側に知られ、「初心者向けの導き手」という役割を担うようになりました。47歳の遅参者だからこそ「失敗を経験した上での説明」が、初心者に刺さるのだと気づきました。
