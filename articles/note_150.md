# 【AI対話アプリ】200の問い #150「Eスポーツ・戦術分析」

このアプリは「Eスポーツ・戦術分析」のためのAI対話プロンプト集です。ゲーム動画を「単なる上手さ」から「戦術の深さ」へと読み替え、プロ選手の思考を言語化して、自分の上達に応用します。

---


## 1. プロ選手の「1つの動き」を「3段階の意思決定」に分解する

見た目は「キャラが走ってジャンプした」だけに見えても、その背景には複数の判断があります。AIに「プロ選手のこのプレイ動画の『5秒間』について、『なぜその行動をしたのか』という3段階の意思決定を推測してください」と頼みます。「1段階目:敵がここに出現する可能性が高い 2段階目:先制攻撃を取るために高所を確保する必要がある 3段階目:このルートなら敵に見つからずに高所に達する」という分解があると「うまさ」は「戦術的判断」に翻訳されます。


## 2. 「失敗した瞬間」に隠れた「判断ミス」を言語化する

プロ選手が敗北する瞬間、その背景には「戦術的誤判断」があります。AIに「この試合で敗北する1分前の動き について、『ここで別の選択肢があったはず』という3つの分岐点を挙げてください」と頼みます。「敵の位置情報を過信した」「リソース管理を見誤った」「ポジショニングの遅延」という言語化があると「負け」は「学習機会」に変わります。


## 3. 「自分のランク」と「プロのランク」のギャップを戦術で測る

「俺はプロより下手」という評価は曖昧です。AIに「あなたの直近ゲームで、プロ選手なら『ここで別の判断をする』という瞬間を5つ挙げてください。その判断の違いは、スキルですか、それとも戦術的読みですか」と頼みます。ギャップが「反応速度」ではなく「敵の出現予測」にあれば、上達の戦略が変わります。


## 4. 「チームゲーム」での「連携の深さ」を役割に細分化する

eスポーツ、特にチームゲームは、連携が勝敗を決めます。AIに「このチームが敵を一掃した5秒間について、5人の選手が『何を想定して、どう動いたのか』を役割ごとに説明してください」と頼みます。その連携の意図が言語化されると、自分たちのチーム内でも「同じ意図を再現する」訓練ができます。
