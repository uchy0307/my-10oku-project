# 【AI対話アプリ】200の問い #152「自由研究・テーマ探し」

このアプリは「自由研究・テーマ探し」のためのAI対話プロンプト集です。子どもの「日常の疑問」から「研究テーマ」へ昇華させ、親子で取り組む探究学習へ導きます。

---

47歳の私は、小学5年生の息子から「自由研究、何したらいい?」と聞かれ、答えられませんでした。妻は「何か興味あることやったら?」と言いましたが、息子は「特にない」と答えました。その夜、妻が「子どもって、実は疑問だらけなんだけど、『研究テーマにする』という言語化ができてないのかも」と指摘しました。確かに、息子は毎日「なんで?」「どうして?」と聞きます。その日常の疑問を、研究テーマに変換する「プロセス」があれば、自由研究は苦痛ではなく「冒険」になるのではないか。そう考えて、AIとの対話が始まりました。

## 1. 日常の「なぜ?」を「研究仮説」に昇華させる

子どもの疑問は、しばしば曖昧です。AIに「この子が『なぜ、蝶は色が多いの?』と聞いたとき、その裏にある『本当の疑問』は何か推測してください。そしてその疑問を『検証可能な仮説』に変えてください」と頼みます。「『蝶は色が多い』→『蝶の色は、捕食者からの防御に関係しているのでは』」という変換があると、子どもは「あ、自分の疑問は『研究』になるんだ」と気づきます。

## 2. 「仮説の検証方法」を「遊び」として再設計する

子どもが「研究」と聞くと、難しいと感じます。AIに「この仮説を確かめるための方法を、『楽しい遊び』のように見える3つの方法に言い換えてください」と頼みます。「色の異なる蝶の標本を集める」は「蝶採集ゲーム」、「捕食者を模した実験」は「蝶の天敵シミュレーション遊び」という言い換えがあると、子どもは「面白そう」と感じます。

## 3. 「親としての関わり方」を具体化する

自由研究は「子どもがやること」ですが、親の関わりが成否を決めます。AIに「この研究を進める親として、『助言してはいけないこと3つ』『むしろ質問すべきこと3つ』を挙げてください」と頼みます。「助言してはいけない:答えを教える / 結論を誘導する / 親の興味優先」「質問すべき:何で『その方法』を選んだ? / 結果をどう解釈する? / 次は何を知りたい?」という対比があると、親は「支援者」に徹することができます。

## 4. 「研究成果の表現方法」を柔軟に用意する

研究成果は「レポート」に限りません。AIに「この研究結果を『レポート』『ポスター』『動画』『プレゼン』の4種類で表現するなら、それぞれどう違う形になりますか。子どもが『これなら楽しく表現できる』と感じる形はどれですか」と頼みます。その「選択肢の提示」があると、子どもは「自分にあった表現方法」を選べます。

## 体験談: 「テーマ選びで3週間止まってた」から「完成まで4週間」の話

正直に言うと、息子は「研究テーマ選び」で、夏休みの最初の3週間を浪費していました。毎日「何にしようかな」の繰り返しでした。その時点で、AIに「この子の『日常の疑問』を全部挙げてください。そしてそれぞれを『研究仮説』に変えてください」と頼みました。出てきたのは、5つの候補でした。その中から、息子が「これなら面白そう」と選んだのが「蝶の色」でした。その後、AIに「この仮説を検証する『楽しい遊び的方法』3つ」を聞くと「蝶採集」「捕食者シミュレーション」「色の心理実験」という3つが提示されました。息子はそれを全部やることにしました。そして4週間、父子で蝶採集に出かけ、捕食者役になってシミュレーション遊びをし、友人たちに「蝶の色、どれが見えやすい?」という実験をしました。最後は「動画での表現」を選び、スマホで撮影した映像を組み立てて、YouTube的な形式で「研究結果」を表現しました。学校へ提出したレポートではなく、その動画が「親友たちの間でバズ」して、別のクラスの子からも「あの蝶の研究の子?」と声をかけられたそうです。自由研究は「学校の課題」ではなく「子どもが『自分の疑問を世界に表現する』最初の経験」だったのです。

- ❌ 30日で焦らない（深い変化は90日後に表面化する）
