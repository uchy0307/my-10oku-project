# 【AI対話アプリ】200の問い #155「趣味記録・キャンプ」

このアプリは「趣味記録・キャンプ」のためのAI対話プロンプト集です。キャンプ日誌や趣味の日々の記録を「単なる備忘録」から「人生の物語」へ昇華させ、思い出を深める自己省察を促します。

---


## 1. 「その日の出来事」を「人生のテーマ」に接地させる

日誌は「朝8時に起床、焚き火をして…」という事実の列挙で終わることが多いです。AIに「このキャンプで起きたエピソード3つについて『これは、あなたの人生の『何の課題』を象徴しているか』を考え、それぞれを1文で言語化してください」と頼みます。「子どもが『火の使い方を教えて』と頼んできた→ 『親から子への知識の継承』という人生テーマ」という接地があると、記録は「単なる日誌」から「人生の通史」に変わります。


## 2. 「家族の『小さな変化』」を見つけて言語化する

キャンプで気づく家族の変化は、日常では見逃されます。AIに「このキャンプで『家族のメンバーの小さな成長や変化』5つを挙げてください。写真には映らない『心の動き』として」と頼みます。「妻が『私も焚き火で何か作ってみたい』と言った→ 普段の家事から『共同作業への意欲』への変化」という気づきがあると、記録は「その家族の歴史」になります。


## 3. 「自分自身の『気づき』『学び』」を3段階で深掘りする

その場の感動は、後日、「なぜ感動したのか」という問いで、より深まります。AIに「このキャンプで『ああ、人生ってこういうことなんだ』と感じた瞬間は? その瞬間を『現象レベル』『心情レベル』『人生観レベル』で3段階掘り下げてください」と頼みます。「朝日を見た(現象) → 美しさに息をのんだ(心情) → 『毎日忙しく走っている自分は、こういう瞬間を失っている』(人生観)」という3段階があると、記録は「自己省察」になります。


## 4. 「次のキャンプへの『未完了な問い』」を記す

記録は、完結ではなく「次への道筋」を示すべきです。AIに「このキャンプを終えて『次のキャンプで、ぜひ試してみたいこと』『まだ確認できていない疑問』『家族とやり残したこと』を3つ書いてください」と頼みます。その「未完了な問い」が、次のキャンプへの期待や、人生の探究の継続性を作ります。
