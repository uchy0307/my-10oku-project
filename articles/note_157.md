# 【AI対話アプリ】200の問い #157「写真整理・物語化」

このアプリは「写真整理・物語化」のためのAI対話プロンプト集です。スマホの中の「ランダムな写真の集まり」を「人生の通史」として物語化し、思い出を深め、家族との絆を可視化します。

---


## 1. 「時系列に写真を並べる」ことで「人生のテンポ」を見つける

単に「写真をフォルダ分け」しても、意味がありません。AIに「スマホの最新100枚の写真を『時系列に眺めたとき』『あなたの人生は今、どういうテンポで進んでいるのか』を説明してください」と頼みます。「仕事の写真が増えている → 仕事の負荷が上がっている時期」「家族の写真が減っている → 親たちとの時間を失っている」という「テンポの読み」があると、ただの「整理」が「人生への省察」に変わります。


## 2. 「写真から『その時の物語』を復元する」

写真は「その瞬間」しか記録しません。でも、前後の文脈があると「物語」が立ち上がります。AIに「このキャンプの写真10枚を見て『その週末、何が起きていたのか』『誰が、どう感じていたのか』という『物語』を復元してください」と頼みます。その復元が「子どもたちへの『思い出の語り部』」になります。


## 3. 「1年分の写真」を「15枚に厳選」して「その年の物語」に凝縮する

スマホには膨大な写真がありますが、「本当に大事な瞬間」は限られています。AIに「この1年間の写真から『その年を象徴する15枚』を選んでください。それぞれのカットに『その写真が示しているテーマ』を1文で付けて」と頼みます。その「15枚の厳選」が、1年という時間の「本質」を映し出します。


## 4. 「物語化された写真」を「プレゼンテーション」で家族と共有する

整理した写真は「見せる」ことで初めて生きます。AIに「この15枚の写真を『親に見せるプレゼンテーション』として『物語の流れ』を組立ててください。説明は『親が涙ぐむ』レベルの感動を狙って」と頼みます。その「プレゼンテーション」が、親たち(祖父母)の心を揺さぶります。
