# 【AI対話アプリ】200の問い #163「塾・問題解決の喜び」

このアプリは「塾・問題解決の喜び」のためのAI対話プロンプト集です。生徒が「問題が解ける」という経験の先にある「自分の思考回路が動いている実感」をAI対話で可視化し、テスト成績ではなく「問題解決のプロセス喜び」を教育の中核に据えます。

---


## 1. 「問題が解けない」を「どこまでわかっているか」に言い換える

生徒が「わかりません」と言うとき、その背景には「完全にわからない絶望」ではなく「部分的にはわかっているが、次が見えない」という状態があります。AIに「この問題について、生徒が『実は理解していることは何か』を5段階に分解して。完全な理解から、かすかな理解まで」と依頼します。例えば数学の連立方程式で「わかりません」と言う子でも、「×の係数を消す」というアイデアまでは持っていたりします。その「部分的な正解」を言語化することで、生徒の心が一気に軽くなります。「君はここまではちゃんと考えてる」という一言が、恐怖を好奇心に変えます。


## 2. 「試行錯誤の履歴」を褒める文化を作る

学校教育では「正解」だけが評価され、間違いは無視されます。これが生徒の思考を奪います。AIに「この生徒が『間違えた計算過程』から学べることを、教育学的に3点指摘してください」と依頼します。生徒に返す言葉は「ここの符号は違うけど、この考え方は本当に良い」という間違いの中の正解を見つけることです。3ヶ月続けると、生徒は「間違いは悪」ではなく「間違いは思考の痕跡」という認識が育ちます。その時点で、学習速度が劇的に変わります。


## 3. 「問題を自分で作る体験」で、出題者の思考に入る

問題集を解くだけでは、問題の構造が見えません。AIに「この生徒が学んだ概念を使って、友達が『最初はわからないが、解けたら嬉しい』という問題を、自分で1問作ってください」と依頼します。その問題を作るプロセスで、生徒は出題者の視点に立ちます。すると、テストの問題も「誰かが仕込んだワナ」に見えるようになり、読み方が深くなります。


## 4. 「解けたあとの満足感」を定量化して、次への動機づけにする

テスト直前の生徒は「失敗への恐怖」に支配されています。AIに「このテストで正解したとき、この生徒が『自分がどう成長したか気づく質問』を5問作って」と依頼します。テスト後に「点数」ではなく「君のこの部分が本当に強くなった」と質問を通じて確認させることで、次のテストへの動機づけが、恐怖ではなく好奇心に変わります。


## 体験談: 「わかりません」と5分間言い続けた男子が、2ヶ月後に『問題を教えてくれる側』になった話

その男子は最初の授業で、本当に「わかりません」と5回以上言いました。そこから始めたのは、彼が「何をわかっているか」の徹底的な言語化でした。初月は彼がどの問題でも「わかっていること」を見つけることだけに注力しました。2月目に入ると、彼から「先生、ここで僕が詰まるのはなぜ？」という質問が初めて出ました。その質問を聞いた瞬間、私は感激しました。彼は「わかりません」ではなく「わかりたい」へ変わっていたのです。3ヶ月後、そのテストの成績は42点から78点に上がっていました。しかし、最も嬉しかったのは、彼が友人に「この問題、こう考えると面白いよ」と説明している光景です。恐怖が好奇心に変わった瞬間、その子は教える側に立っていました。
