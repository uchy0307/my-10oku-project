# 【AI対話アプリ】200の問い #165「カフェ・マイクロコピー」

このアプリは「カフェ・マイクロコピー」のためのAI対話プロンプト集です。顧客が飲食店で「迷う瞬間」の心理状態をAI対話で言語化し、その0.5秒の迷いを『たった一行の言葉』で決断へ導くメニュー文を設計します。

---


## 1. 「メニューを見ている顧客の心の問い」をAIで可視化する

顧客がメニューを見ているとき、彼らは「飲み物を選ぶ」のではなく「今、自分に必要な体験を選ぶ」という無意識の問いに答えています。AIに「このカフェの顧客が『アイスコーヒーか、アイスラテか』で迷うとき、その心の中では『どんな問い』が起こっているか。その問いは『味の違い』ではなく『その飲み物が与えてくれる〇〇の違い』か、3パターン想像してください」と依頼します。例えば「目覚めの爽快感か、深い思考の落ち着きか」「素早くエネルギーがほしいのか、ゆっくり味わいたいのか」。その心の問いが見えると、メニュー説明が変わります。


## 2. 「顧客の時間帯」に応じた『推しフレーズ』を設計する

同じコーヒーでも、朝8時と午後3時で、顧客の心は異なります。AIに「このカフェの顧客が『午前中に立ち寄るとき』『午後に立ち寄るとき』『夜間に立ち寄るとき』で、それぞれ『このコーヒーを飲んでいるあなたは、どんな状態になるか』を20字以内で表現してください」と依頼します。その表現をメニューに「朝: 脳が目覚める一杯」「午後: 思考が冴える瞬間」と書き込むだけで、顧客の選択が変わります。


## 3. 「感情キーワード」を抽出して、メニュー説明に仕込む

商品説明は「豆の種類・焙煎度」ではなく「この飲み物があなたにもたらす感情」を言語化すべきです。AIに「このメニュー項目を『選んだ人がその飲み物を飲んでいるとき、どんな感情を感じるか』を3つ、各5字で抽出してください」と依頼します。例えば「爽快感・深思・余韻」といった感情キーワードが出たら、メニューに「朝の一杯。爽快感を。」と仕込みます。その一行が、迷いを決断に変えます。


## 4. 「前の客の選択」を『信号』にする

心理学の「バンドワゴン効果」を使い、AIに「このコーヒーを選ぶ人の『プロフィール』を3タイプ作って。年代・職業・時間帯・心理状態を含めて」と依頼します。そしてメニューに「営業職の朝1杯。目覚めの相棒。」と、『誰が選ぶのか』を書き込みます。顧客は「あ、自分と同じタイプの人がこれ選ぶんだ」と安心し、迷いが消えます。


## 体験談: メニュー文を変えた1週間で、客単価が12%上がった話

その女性に「アイスコーヒーですね」と言った後、私はメニュー全体を3日かけて書き直しました。AIとの対話で、顧客の「迷い」を全30商品について可視化し、それぞれに「この飲み物があなたにもたらすもの」を20字以内で書き込みました。例えば、従来の「アイスラテ \600」は「深い思考の友。ミルクの優しさで、落ち着きを。」に変わりました。変化は即座に現れました。初日から、複数商品を注文する顧客が増えました。「アイスコーヒーください」だけでなく「このコーヒーと、パニーニもください」という連動購買が、1週間で30%増加しました。最も驚いたのは、リピート率です。同じ顧客が1週間に3回来店するようになり、毎回違う商品を選んでいました。その女性も再来店し、今度は迷わずに「今日は、思考の友をください」と注文してくれました。
