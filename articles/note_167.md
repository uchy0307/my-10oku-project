# 【AI対話アプリ】200の問い #167「占い・未来の選択肢」

このアプリは「占い・未来の選択肢」のためのAI対話プロンプト集です。顧客が人生の分岐点で迷うとき、AI対話で『複数の未来ビジョン』を言語化し、選択肢を視覚化させながら、自分自身の「本当の望み」に気づかせます。

---


## 1. 「選ばなかった方の未来」を詳細に想像させる

人は「選んだ方」の未来しか見ません。AIに「この男性が『転職しなかった場合の3年後』を想像してください。仕事・給与・心情・人間関係・自分への評価が『どう変わったか』を、各300字で」と依頼します。多くの人は「選ばなかった方の悔しさ」を想像します。でも実際には「現職で深くなる喜び」があったりします。その喜びを言語化することで、選択は複雑になり、本当の「理由」が見えます。


## 2. 「5年後、どちらを選んでいても『幸せ』か」を確認する

短期的には転職が有利でも、5年後に同じ充足感があるか不明です。AIに「『転職した場合の5年後』と『現職を続けた場合の5年後』で、この男性が『心底満足している』のはどちらか。その理由まで」と依頼します。多くの場合、答えは「選択肢そのものではなく『その選択をした自分の努力と成長』に満足している」というパターンです。


## 3. 「選択の理由」を『義務』から『欲求』に変える

「転職した方が得」という外部的理由で選ぶと、後悔が生まれます。AIに「この男性が『転職を選ぶとしたら、最も自分を動かす理由は何か』。給与ではなく『どんな自分になりたいのか』」と依頼します。その問いが答えられない場合、その選択は本当の選択ではなく、外部圧力への追随です。


## 4. 「選択後に起こる後悔」を事前に言語化する

どちらを選んでも、後悔は生まれます。AIに「『転職を選んだ場合の後悔シーン3つ』『現職を続けた場合の後悔シーン3つ』を想像してください」と依頼します。その後悔を事前に知ることで、「その後悔に耐えられるか」という基準で選べます。


## 体験談: 「転職すべき」と言わず、3時間の対話で『本人が決めた』話

その男性との対話は3時間続きました。最初は「転職するべき？」という質問でしたが、段々と「なぜ転職したいのか」「現職で何が足りないのか」という本質的な問いに変わっていきました。2時間目に、彼は泣きました。「給与のためじゃなく、自分が『成長できていない』という焦りがある」と気づいたのです。3時間目、彼は「転職します。でも、後悔も覚悟しています」と言いました。占い師の私は「どちらが正解」とは言いません。その男性が「自分の選択として引き受けた」その瞬間が、本当の選択だと知っていたからです。

3. この選択で「後悔すること」は何か
4. その後悔に「耐えられるか」
5. 10年後、この選択について何と言っていたいか
