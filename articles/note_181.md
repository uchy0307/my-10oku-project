# 【AI対話アプリ】200の問い #181「求人・魅力の再定義」

このアプリは「求人・魅力の再定義」のためのAI対話プロンプト集です。採用面接で「御社で働きたい理由は?」と聞かれて詰まる求職者の悩みを逆転させ、企業側が「この人を採らなきゃ」と言わしめる自分の市場価値を言語化します。

---


## 1. 「経歴」と「属人的スキル」を分離する

多くの転職者は、履歴書に書いた「◯◯会社で部門長」を誇りに思っています。それは間違いです。その会社があってこその肩書だからです。AIに「今までの仕事で、会社と無関係に自分だけが持ち運べるスキルは何か、5つ挙げて」と頼みます。私の場合、「月商1000万の営業チーム構築」は属人的スキルではなく「その組織における肩書」でした。本当のスキルは「5段階の昇進要件を逆算して、5年で人を育成する人事設計」でした。前者は書類で落ちる。後者は面接官の目が変わります。


## 2. 「失敗から学んだこと」をAIに言語化させる

新しい環境で活躍する人の共通点は「失敗を再解釈できる」ということです。AIに「あなたのキャリアで一番後悔している決定を1つ挙げ、その失敗があなたにもたらしたもの3つを推論してください」と頼みます。私は「部下のAさんを引き継ぎなしに別部門に配置転換したこと」を挙げました。AIが返してきたのは「1.権限のない環境での説得スキル 2.失敗後の立て直し経験 3.人間関係修復の工夫」でした。これを面接で語ると、人事は「単なる失敗の言い訳」ではなく「失敗をどう吸収する人間か」を評価します。


## 3. 「応募先企業が実は求めている人物像」を逆算する

募集要項は建前です。採用面接官の本音は「この部門の実績を上げるために、今いない人物像」です。AIに「〇〇業界で××職の募集を見たとき、募集文に書かれていない『本当に必要な3つの属性』を推測してください」と頼みます。例えば「営業部次長」の募集なら、募集文に「数字に強い」と書かれていても、本当の採用基準は「既存営業との確執を調整できる人間力」かもしれません。自分がそれを持っているなら、そこに焦点を絞ります。


## 4. 「給与交渉の根拠」を数値化する

年収は「希望額」で決まりません。「自分が企業にもたらす価値」で決まります。AIに「あなたの過去10年で、数字で測定できる成果を5つ挙げて、その企業にとっての換算価値を計算してください」と頼みます。「営業部を率いて年商30%増」なら、それが新企業でも30%増なら年間3億円の売上増。給与の根拠は「月給40万円」ではなく「年間3億円の価値を安定供給できる人間だから、年600万円は妥当」という逆算になります。


## 体験談: 「肩書で落ちた」から「属人性で受かった」へ

正直に告白します。私は5社連続で一次落ちを経験しています。理由は簡単で「前職での肩書を言い続けた」から。6社目の面接で、AIの指導を受けて「前職では、新人と責任ある立場のギャップを埋めるために、独自の育成フレームを作りました」と言いました。その瞬間、面接官が身を乗り出しました。「それ、うちでも困ってた」と。給与交渉の際も「年間これだけの人的資産を作れるから」と数値で説明しました。結果、提示額より月10万円上が確定しました。肩書を手放すことで、初めて自分が見えました。

## 🎯 上級プロンプト: 「面接官の本音ヒアリング」法

通常の自己分析で詰まったら、AIに次のように頼みます:
「採用面接官が、募集文には書かない『本当に困っていること』を言語化してくれるなら、あなたはそれを聞き出す質問を5つ設計してください。」

```
あなたは、求職者の市場価値を最大限に引き出し、企業が採用したくなる魅力を言語化するキャリアコーチです。以下の問いに具体的に回答してください。あなたの回答を元に、市場価値を再定義し、強力なアピールポイントを見つけ出すお手伝いをします。

まず、これまでの職務経験で得たスキル、達成実績、担当業務を全て書き出してください。数値で表せる成果は必ず含めてください。

次に、書き出した中で「他者には真似できない、苦労して習得した独自性の高いもの」は何ですか？どのような状況で役立ち、どんな困難を乗り越えた経験と結びついていますか？具体例を交えて説明してください。

その独自性の高い経験やスキルが、企業が抱える「人材不足」「新規事業」「既存事業の改善」といった課題を解決するとしたら、どのように貢献できますか？具体的な貢献イメージを複数提示してください。

同業他社の求職者と比較して、あなただけが提供できる「独自の価値」や「強烈な個性」は何だと思いますか？なぜそう言えるのか、根拠と共に説明してください。

上記の対話を踏まえ、「この人を採らなきゃ」と企業が強く感じるような、あなたの「唯一無二の魅力」を簡潔なキャッチコピーと具体的な説明で表現してください。

上記の対話を通じて明らかになった市場価値を最大限に活かすため、今後のキャリア形成や求職活動で取るべき次の一歩を3つ提案してください。
```

このプロンプトの威力は、応募先企業の『実は抱えている課題』を事前に把握できることです。私はこれで「営業育成の内製化が急務だった」という本音を引き出し、面接で「その内製化に必要な4つのステップ」を提案しました。合格です。
