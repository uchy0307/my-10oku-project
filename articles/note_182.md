# 【AI対話アプリ】200の問い #182「就活・ガクチカ深掘り」

このアプリは「就活・ガクチカ深掘り」のためのAI対話プロンプト集です。「学生時代、頑張ったことは?」という面接の定番質問で「起業しました」「留学しました」と言う代わりに、その経験の「本当の学び」を多層化させて、面接官が「この子は本質を見ている」と評価するレベルに引き上げます。

---


## 1. 「経験の表面」から「その経験が変えたもの」を3段階掘る

就職活動において、単に「何を頑張ったか」を伝えるだけでは不十分です。このアプリでは、あなたの経験を「表面的な事実」「そこから得た学び」「その学びが自分自身をどう変えたか」という三段階で深く掘り下げます。

まず、第一段階は「何をしたか」という事実の確認です。例えば、AIに「私がプロジェクトで取り組んだことについて、まず『具体的な活動内容』を教えてください」と問いかけます。次に第二段階として「その経験から、何を学び、どんなスキルが身についたか」を深掘りします。ここで得られるのは、具体的な業務スキルや課題解決能力といった教訓です。

そして最も重要なのが第三段階。「その学びは、あなたの考え方や行動、価値観にどんな変化をもたらしましたか？」と問いかけることで、経験があなた自身をどう変容させたのか、本質的な部分に迫ります。私自身も、この問いによって、以前は単なる「成功体験」と捉えていた出来事が、実は自分の人生観を形成する大きな転換点だったと気づかされました。このアプリは、面接官が求める「本質を見抜く力」を育むための強力なツールとなるでしょう。


## 2. 「失敗の再解釈」をAIが手伝う

私たちは誰もが失敗を経験します。しかし、その失敗を単なるネガティブな出来事で終わらせてしまっていませんか？このアプリでは、AIがあなたの失敗経験を多角的に分析し、そこから得られる真の学びや成長の機会を浮き彫りにします。例えば、「プロジェクトの失敗で、私は具体的に何を学ぶべきだったのでしょうか？」や、「あの時の判断ミスが、今の私の価値観にどう影響していますか？」といった問いかけをAIにしてみてください。

私自身、過去の失敗を語る際、「反省しました」で終わってしまうことが多く、それが面接官に響く言葉ではないと悩んでいました。しかし、AIとの対話を通じて、表面的な反省だけでなく、その失敗が自分の内面にどのような変化をもたらし、結果として現在の強みや視点に繋がっているのかを深く掘り下げることができたのです。

AIは、あなたの言葉の中から本質的な課題や潜在的な能力を見つけ出し、失敗が実はかけがえのない成長の糧であったことを再認識させてくれます。単なる経験談ではなく、その背後にあるあなたの人間性や本質的な思考力を面接官に伝えるための強力なツールとなるでしょう。失敗をポジティブな財産に変え、自信を持って語れるようになるためのサポートをAIが提供します。


## 3. 「同年代ではなく、社会人目線で」自分を説明する言葉を作る

学生同士の会話と、面接は違います。AIに「あなたの経験を、同じ経験をしたことのない40代の責任ある立場に説明するなら、どんな『一般化された学び』として説明しますか。具体例を除いた『原則』の形で5つ言語化してください」と頼みます。「サークルで部員が50人に増えた」という事象を、「組織拡大期には、新規採用と既存メンバーの満足度維持の両立が課題になり、その課題を両立させるプロセス」と言い換えることで、新入社員としての価値が見える。年功者にとって分かる言葉で自分を説明できる人は、入社後も「この子は成長する」と評価されます。


## 4. 「その学びが、応募先企業でどう生きるか」を具体化する

ガクチカの最後の落としが、面接官の全体評価を決めます。AIに「あなたが学んだ『この3つの原則』が、応募企業の〇〇職で、初年度のどんなシーンで活躍しますか、3シーンをリアルに描写してください」と頼みます。具体性が生まれます。「報告の前に相手の状況を察する」という学びが、営業職で「顧客の潜在ニーズを先に想像する」というスキルに転化する、という接続が見える人物は、採用面接官から「こういう人材を育てたかった」と思われます。

## 🤔 5つの問い (このアプリの中核)

1. あなたが一番変わった経験は何ですか(変わる前後の違いを明確に)
2. その経験で、あなたは自分の何を「勘違いしていた」ことに気づきましたか
3. その気づきがなかったら、あなたはどんな人間のままでしたか
4. 応募先企業で初めて「困ることになる場面」を3つ想像してください
5. その困りごとに対して、学生時代の「あの経験」がどう役に立つか説明できますか
