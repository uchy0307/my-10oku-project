# 【AI対話アプリ】200の問い #182「就活・ガクチカ深掘り」

このアプリは「就活・ガクチカ深掘り」のためのAI対話プロンプト集です。「学生時代、頑張ったことは?」という面接の定番質問で「起業しました」「留学しました」と言う代わりに、その経験の「本当の学び」を多層化させて、面接官が「この子は本質を見ている」と評価するレベルに引き上げます。

## 2. 「失敗の再解釈」をAIが手伝う


## 3. 「同年代ではなく、社会人目線で」自分を説明する言葉を作る

学生同士の会話と、面接は違います。AIに「あなたの経験を、同じ経験をしたことのない40代の経営層に説明するなら、どんな『一般化された学び』として説明しますか。具体例を除いた『原則』の形で5つ言語化してください」と頼みます。「サークルで部員が50人に増えた」という事象を、「組織拡大期には、新規採用と既存メンバーの満足度維持の両立が課題になり、その課題を両立させるプロセス」と言い換えることで、新入社員としての価値が見える。年功者にとって分かる言葉で自分を説明できる人は、入社後も「この子は成長する」と評価されます。


## 4. 「その学びが、応募先企業でどう生きるか」を具体化する

ガクチカの最後の落としが、面接官の全体評価を決めます。AIに「あなたが学んだ『この3つの原則』が、応募企業の〇〇職で、初年度のどんなシーンで活躍しますか、3シーンをリアルに描写してください」と頼みます。具体性が生まれます。「報告の前に相手の状況を察する」という学びが、営業職で「顧客の潜在ニーズを先に想像する」というスキルに転化する、という接続が見える人物は、採用面接官から「こういう人材を育てたかった」と思われます。


## ✅ 4項目セルフチェック

- [ ] 1つのガクチカを「困りごと → 試行 → 成長」の3段階で深掘りした
- [ ] 失敗経験を「その失敗が教えてくれたこと」として言い換えた
- [ ] 「同年代向けの説明」ではなく「社会人向けの原則化」ができた
- [ ] その学びが「応募企業のどのシーンで活躍するか」を3つ具体化した


## 🤔 5つの問い (このアプリの中核)

1. あなたが一番変わった経験は何ですか(変わる前後の違いを明確に)
2. その経験で、あなたは自分の何を「勘違いしていた」ことに気づきましたか
3. その気づきがなかったら、あなたはどんな人間のままでしたか
4. 応募先企業で初めて「困ることになる場面」を3つ想像してください
5. その困りごとに対して、学生時代の「あの経験」がどう役に立つか説明できますか
