# 【AI対話アプリ】200の問い #184「配送・プロの気配り」

このアプリは「配送・プロの気配り」のためのAI対話プロンプト集です。配送業界で「単なる物運び」から「顧客信頼の構築者」へ昇華するために、配送員が日々出会う「予期しない現場」で取るべき判断と、その判断の背景にある「顧客心理の読み方」を言語化します。

---

## 1. 「配達に来る意味」を顧客の人生に繋ぎ込む

配送員の9割は「パッケージを玄関に置く」で仕事が終わると思っています。実際は違います。AIに「今日の配達先リストで、独居の高齢者や身体が不自由な方を想定して、『その方にとって、あなたが来ることの意味は何か』を1件ごとに想像してください」と頼みます。例えば「毎週木曜に通販で生活用品を注文する70歳」なら、配達の意味は「物を受け取ることではなく『週に1度、誰かが玄関に来てくれる』という確認」かもしれません。その想像が見えると、配達の姿勢が変わります。ただ置くのではなく「お疲れ様です」の一言が入ります。

## 2. 「トラブルの前兆」を現場で読む力

配達トラブル(受け取り拒否、返品、クレーム)の9割は「最初の訪問時の『何か』が原因」です。AIに「以前『トラブル宛先』だった場所を2件思い出して、その時『どんな受け答えをしていたか』『相手はどんな反応をしていたか』を描写してください。その反応は『物が欲しくない』という意味だったか『信頼がない』という意味だったか、AIに仕分けてもらってください」と頼みます。私の場合「玄関に出てこない高齢者」は「外出が億劫」だったのではなく「配達員の言葉が早すぎて、不信を感じていた」ことに気づきました。前兆を読める人間は、トラブルを未然に防げます。

## 3. 「判断の迷い」をAIに相談する習慣

配達現場では、毎日「置いて帰っていい?」「玄関じゃなく裏口?」「サインなしで置く?」という判断が迫られます。マニュアルと現実にズレがあります。AIに「こういう状況が来たら、あなたならどう判断しますか」と事前に相談する習慣。例えば「不在で、置き配指定なし、高額商品」という状況なら、AIが「その顧客の購買履歴から判断して」「その地域の防犯状況から」「法的な責任から」という複数角度で「判断の枠組み」を示してくれます。ドライバーの判断は「その枠組みの中で」迷わなくなります。

## 4. 「苦情の本質」を聞き直す力

配送へのクレームが来たとき「荷物を落とした」「遅れた」という表面の苦情ではなく、その裏にある「信頼の破壊」を見ます。AIに「顧客から『遅刻した』というクレームが来たとき、相手が本当に怒っていることは何か、5つの可能性を推測してください。『時間に遅れたこと』以外で」と頼みます。「計画が狂った」「連絡がなかった」「謝罪の言葉がなかった」など、事実とは別の『心理的な失礼』が見える。そこに気づく配送員は、謝罪の仕方が変わります。

## 体験談: 「単なる物運び」から「見守り仕事」への昇華

3年目、私は配送の「効率化」に執着していました。1日50件を時間内に終わらせることだけが目標。なのに「クレーム宛先率」が部内で最も高かった。新人の若いドライバーがこちらの地区を手伝いに来て「お疲れ様です」と必ず言うのを見ていて、初めて気づきました。配送は「走る仕事」ではなく「信頼を運ぶ仕事」だった。それ以降、1日40件に減らし、1件に3分多くかけるようにしました。結果、クレームは消えました。むしろ「この人に配達してもらえると安心」というコメントが、複数の顧客から来るようになりました。配送ドライバーで初めて「やりがい」を感じました。
