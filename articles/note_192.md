# 【AI対話アプリ】200の問い #192「秘書・贈答品センス」

このアプリは「秘書・贈答品センス」のためのAI対話プロンプト集です。「完璧な高級品を選ぶ」という間違った努力から解放し、「相手の『その時の心情』『人生のステージ』『実は欲しいもの』を読む力」で、相手が「え、これ、私のために?」と感動する品選びを言語化します。

---


## 1. 「相手の『ライフステージ』」から「その人が本当に求めているもの」を逆算する

贈り物選びで最初にすべきは「相手が『何が欲しいか』という要望」ではなく「相手が『人生のどの段階にいるか』という解釈」です。AIに「上司は『退職』という人生の大きな転機にいます。その転機で『実は何を求めているか』を『経済的な豊かさ』『時間的な豊かさ』『人間関係的な豊かさ』の3視点から推測してください」と頼みます。その答えが「時間的豊かさ（孫との時間）」なら「高級時計」ではなく「孫と遊ぶ遊具」や「一緒にできる体験」に繋がります。ライフステージが見えると「相手が本当に求めているもの」が浮かびます。


## 2. 「相手が『実は言わない欲しいもの』を現場から読む」

相手が「何も要らない」と言うのは「自分のための高級品は要らない」という意味です。でも「一緒に使えるもの」「家族のためになるもの」には全く違う反応をします。AIに「上司の『日常会話』『行動』『購買履歴』から『実は求めているが言わないもの』を5つ推測してください」と頼みます。「図書館に籠もりたい」という発言から「読書用の高級クッション」「作家の初版本」など、具体的な品が見えてきます。言わない欲しいものを読める秘書は「あ、この秘書は自分を分かっていた」という信頼を生みます。


## 3. 「相手が『その時、心が温かくなる瞬間』を想像する」

贈り物の本当の価値は「品そのもの」ではなく「その品を手にした時『相手の顔がどう変わるか』」です。AIに「上司が『この贈り物』を手にした瞬間、どんな『心理的な反応』が起きるか、3段階で描写してください。①『あ、これ、私のために考えてくれた』という気づき ②『なぜ自分のことをこんなに分かってくれたのか』という感動 ③『この会社にいて良かった』という満足度」と頼みます。その3段階が全て見える品を選んだとき、相手の反応が全く違います。


## 4. 「相手の『次のステージ』を応援する品選び」

退職後、新しい人生が始まります。その新しい人生を「応援する」という視点で品を選び直す。AIに「上司が『定年後の人生で、最初の3ヶ月で実現したいこと』3つを想像して、その実現を応援する『品』を逆算して設計してください」と頼みます。「孫との時間を増やす」なら「一緒に過ごすための道具」。「図書館で本を読む」なら「読書環境を整える品」。品が「応援」に変わった時、その贈り物は「高級さ」ではなく「相手への向き合い方」で評価されます。


## 体験談: 「高級品」から「相手の心を読む品」へ

初年度の秘書時代、私は「退職祝いは『最高の品』を選べ」と先輩から教わり、高級な置き時計を選びました。相手の反応は「ありがとうございます」で終わり。形式的でした。その後10年、様々な退職者を見送りながら、ようやく気づきました。「完璧な品」より「相手が『あ、自分のこと分かってくれていたんだ』と感じる品」の方が、心に残る。今回、AIの指導で「上司の『孫との時間』『新しい図書生活』『夫婦で庭いじり』を応援する3つのセット」に変更しました。贈呈式で、上司は「これ、誰がアイデア出したんですか」と聞かれ「秘書の提案」と返答されたとき、涙ぐんでいました。相手の心を読む力が、秘書の価値を決めます。

## 🎯 上級プロンプト: 「相手の心が温かくなる瞬間を先読みする」法

品選びで迷ったら、AIに次のように頼みます:
「相手が『この品』を手にした瞬間から『その後1週間の使用経験』まで、相手の『心の変化』を3つのエピソード(3つの瞬間)で描写してください。」

```
あなたは、贈答品選びにおいて、相手の心に響く「感動」を最優先に考える、パーソナルギフトコンシェルジュです。高価さではなく、相手の「その時の心情」「人生のステージ」「実は欲しいもの」を深く読み解き、心から喜ばれる品選びを言語化するお手伝いをします。

まず、以下の情報について教えてください。

1. 贈りたい相手はどのような方ですか？ (関係性、性別、年代、大まかな職業や趣味など)
2. 今回の贈答のシチュエーションは何ですか？ (誕生日、お礼、昇進祝い、慰労など、具体的な背景も教えてください)
3. その方の現在の状況や、最近の変化について教えてください。(仕事の忙しさ、プライベートでの出来事、健康状態、心境の変化など、可能な限り具体的に)
4. その方が日頃口にしている願望、困っていること、あるいは「これは嬉しいだろうな」と思うような密かな喜びの種について、何か心当たりはありますか？

これらの情報を踏まえ、私が最適なギフトアイデアを提案し、さらにあなたが次の一歩を踏み出すための具体的な行動を3つ提示します。
```

このプロンプトで「相手の反応を先読みできる品選び」が見える。私はこれで「上司が『孫との庭時間でこの道具を使う』という3つのシーンを想像し、そのシーンでの『上司の笑顔』を見えさせることで、初めて『本当の贈り物』が何かに気づきました。


## 📖 追加体験談・失敗回避リスト

- ❌ AIの回答をそのまま貼って終わらせない（一晩寝かせて自分の言葉に翻訳）
- ❌ 1回の対話で結論を急がない（最低3回、角度を変えて問い直す）
- ❌ 完璧を待たない（70%の手応えで動き、残り30%は実践で埋める）
- ❌ 抱え込まない（週1で家族か仲間に1分進捗を語る）
- ❌ 30日で焦らない（深い変化は90日後に表面化する）
