# 【AI対話アプリ】200の問い #194「新人向け・業界の常識」

このアプリは「新人向け・業界の常識」のためのAI対話プロンプト集です。新入社員が「やること」「ルール」という浅い理解で終わらず、「その業界が『なぜ』そう動くのか」という深い構造を理解する30日の対話設計を言語化します。

## 1. 「ルール」ではなく「その業界が『なぜ』そうするのか」を言語化する

新人教育の最大の失敗は「ルールを『ルール』として教えること」。すると新人は「ルールを守る」という低階の思考で仕事をします。AIに「『顧客への報告は毎週』というルール。その奥に『顧客が『信頼を感じる頻度』『市場の変化への対応速度』『営業信頼の構築』という3つの理由」が隠れていないか、を推測してください」と頼みます。ルールの「奥」が見えると「新人は『報告は義務』ではなく『信頼を作る行為』」として仕事をします。

## 2. 「先輩の行動」を「業界の価値観」で解釈させる

新人に「先輩の仕事ぶり」を見せても「真似」で終わります。AIに「営業が『顧客訪問の前に資料を3回見直す』という行動。この行動は『何への怖れ』『何への尊敬』『何への誠意』から来ていると思いますか。その『その行動の奥にある心』を5段階で掘り下げてください」と頼みます。「3回見直す＝顧客の時間を無駄にしたくない」という本質が見えると「新人も『見直しは面倒』ではなく『相手を尊重する行為』」として捉えるようになります。

## 3. 「ミス」を「業界の学習チャンス」に変える

新人のミスを「叱る」ではなく「そこから『業界の価値観』を学ぶ対話」に変える。AIに「新人が『顧客への報告を1日忘れた』というミス。このミス の奥に『この業界で何が起きるか』『なぜ『信頼』が最優先なのか』を説明するような『悔しさを伝える対話』を設計してください」と頼みます。「1日の遅れ＝顧客は『忘れられた』と感じ、信頼が一瞬で崩れる」という『業界の本当の厳しさ』が見えると「新人の『すみません』は『ルール破り』ではなく『相手を傷つけた』という本質的な後悔』に変わります。

## 4. 「30日の終わりに『業界宣言』をさせる」

30日の新人教育の締めくくりは「この業界で『自分が守りたい価値観』を言語化させる」ことです。AIに「新人に『あなたが、この業界で働く理由は何ですか。この業界の『何を守りたいですか』という問いを投げて、その答えを『新人の誓い』という形で1分間語らせてください。30日で学んだ『業界の本質』が『自分の価値観』に翻訳される瞬間」と頼みます。その瞬間「新人は『会社のルール』ではなく『自分たちの業界への誓い』を抱くようになります。

## 体験談: 「『ルール説明3日』から『業界理解30日』へ」

昨年、3日の新人教育で「ルール説明」をして、新人に任せたら「3ヶ月後のアンケートで『まだ『何のためにこうするのか』が分からない』という感想」が並びました。その時、ようやく気づきました。「新人教育＝ルール暗記」では「マニュアル人間」を作っているだけ。「新人教育＝業界の価値観体感」に変えてみました。最初の30日を「ルール』ではなく「なぜ」を問い続ける対話」に変えたところ、3ヶ月後「この業界の仕事『楽しい』」というコメントに変わりました。さらに半年後「自分の判断で『顧客を優先する決定』ができるようになった」という成長報告。ルール「の奥」に「価値観」が見えた新人は、環境が変わっても応用が効く人材に育ちます。
