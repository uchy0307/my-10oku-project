# 【AI対話アプリ】200の問い #195「古着・見極めロジック」

このアプリは「古着・見極めロジック」のためのAI対話プロンプト集です。古着を「古い」「安い」で選ぶのではなく、「その品に『何が詰まっているか』『本当の価値は何か』を読む力」で、他人の「手垢」さえも「物語」に変えて、新しい人生をその服に与えるスキルを言語化します。

---

土曜午後。47歳の古着屋店長である私は、新作入荷の「査定作業」をしていた。革ジャケット、ウールセーター、レーヨン地のワンピース——評価は「状態」「ブランド」「サイズ」の組み合わせだ。「状態8/10、ブランド力5、サイズM」で「査定額5000円」という機械的な算出。だが、売り場で本当に動く古着は「その数値」とは別の「何か」を持っていた。先週、あるお客が「このアクリルのセーター、くたびれてるけど、好きです」と買っていった。セーターは「状態3」「ノーブランド」の商品だった。でも「60年代の編み目の特殊性」「着込まれた色合いの深さ」「その生地が『誰かの20年を見守った』という存在感」が見える人だけが、その価値を読める。査定ロジックではなく「読む力」が、古着の仕事を決めるんだ。今朝、AIに頼むことにした。「古着の『本当の価値』を読む見極める力」について。

## 1. 「状態と年代」から「その服が『何を見守ったか』を推測する」

古着の査定は「状態」で決まると思うのは誤り。本当の価値は「その服が『どの時代』『どんな人生』を見守ってきたか」です。AIに「このセーター。素材はアクリル、色は褪せ、袖の擦り切れが見える。『この擦り切れ方』から『誰が』『どんな人生の中で』『このセーターを何回着たか』を推測してください」と頼みます。ビジネスマンなら「毎日通勤で着込まれた痕」に見えるし、学生なら「週末だけ着た痕」に見えるし、労働者なら「重労働の中での『唯一の温もり』」に見える。その推測が見えると「古い服」ではなく「物語を持つ服」に見えます。

## 2. 「生地の質感」から「時代背景」を読む

古着は「生地」が語る。現代の大量生産品と違い「古着の生地」は「その時代の技術」「その時代の価値観」が詰まっています。AIに「このウール地のセーター。現代のウールより『厚みがある』『毛足が長い』『編み目が詰まっている』。この『3つの特徴』から『このセーターが『いつの時代』『どんな背景』で『誰のために』作られたか」を推測してください」と頼みます。「1960年代、『長く着ることを前提にした高級品』という背景」が見えると、査定額「5000円」では安すぎることに気づきます。

## 3. 「この服を『次の人生』に繋ぐ」という視点

古着屋の本当の仕事は「売る」ではなく「その服を『新しい持ち主の人生に繋ぐ』」ことです。AIに「このセーターが『次の持ち主』のところで『どんな人生を一緒に歩むか』を想像してください。『どんなシーン』『どんな季節』『どんな人間関係』の中で『このセーターが必要になるか』の3つを描写してください」と頼みます。「冬の深夜、一人で考えごとをするときの『温もり』」という使われ方が見えると「単なる古い服」ではなく「その人の『心を守る服』」に見えます。その視点で商品を見ると「配置」「説明文」「おすすめの合わせ方」が全て変わります。

## 4. 「『手垢』を『物語』に変える力」

古着の「汚れ」「褪色」「擦り切れ」——これらは「マイナス要素」ではなく「その服が『生きてきた証拠』」です。AIに「この色褪せたレーヨンワンピース。『この褪色の入り方』『この位置での劣化』から『誰が』『どんな季節』『どんな喜びの中』でこのワンピースを何回着たか」を物語として書いてください。「30代の女性が『毎年夏祭りに着た』という物語」が見えると「褪色」は「欠陥」ではなく「愛用の証」に見えます。

## 体験談: 「査定ロジック」から「物語を読む見極め」へ

最初の5年、私は「状態・ブランド・サイズ」という機械的な査定ロジックで動いていました。するると「売れる商品」と「売れない商品」の差が出ない。どのセーターも「同じ値段」で「同じ扱い」。ある時、新人スタッフが「このセーター、色褪せてるけど『ずっと大事にされてたんだな』という感じがします」というコメント。その新人がその商品を売り場に配置したら「その商品だけ週末に売れた」。その瞬間、気づきました。「古着の価値＝状態ではなく『その服の物語を読む力』」。それからは「査定のときに『この服は誰を見守ったか』という問いを必ず立てる」ようにしました。結果、同じ古着でも「ただの古い服」から「人生を一緒に歩む服」に変わりました。売上も倍になりました。物語を読める古着屋が、選ばれるようになります。

3. その古着を「次に買う人」は、どんな人生を持っていますか
4. その古着が「新しい人生で』何を『守って』あげられますか
5. 古着の「本当の価値」は「状態」ですか、それとも「物語」ですか

## 🎯 上級プロンプト: 「古着の『次の人生』を想像する」法

商品の価値が分からない時は、AIに次のように頼みます:
「このセーター。『状態3/10』『ノーブランド』『時代不明』。でもあなたは『この服には価値がある』と感じた。その『価値』が『次の持ち主のどんな人生』の中で『輝く』か、詳しく描写してください。」

このプロンプトで「査定の目」が「共感の目」に変わります。私はこれで「どのセーターが『誰の心を温める』かが見えるようになり、売り場配置が『状態順』から『物語順』に変わり、その結果『売上が上がった』と同時に『お客さんの『ありがとう』が増えた』」という経験をしました。
