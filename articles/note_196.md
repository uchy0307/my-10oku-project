# 【AI対話アプリ】200の問い #196「投資家・IR要約」

このアプリは「投資家・IR要約」のためのAI対話プロンプト集です。「決算数字」「事業成績」という「事実の羅列」では、投資家の心は動きません。その数字の『奥』にある「経営陣の『覚悟』『未来への確信』『失敗からの学習』」を見える化し、投資家が「この経営陣なら、任せたい」と思わせるIR資料を作る力を言語化します。

---

水曜朝。47歳のベンチャー企業CFOである私は、来月の「投資家向けIR説明会」のプレゼン資料を見つめていた。売上3倍。営業利益黒字化。成長率は業界平均の5倍——数字は完璧だ。でも、資料を眺めていると「なぜ』が足りない。「なぜこの成長ができたのか」「なぜこの失敗から立ち直ったのか」「なぜこの経営判断をしたのか」——その「覚悟の物語」が見えないと「数字だけの企業」に見える。昨年、別のスタートアップの説明会に行ったとき「赤字が続いているが、経営陣の『失敗からの学びの徹底』が見える」という投資家のコメント。その企業に実際に投資が集まったのを見た。数字より「経営陣の『何を信じているか』」が、投資判断を分ける。今朝、AIに頼むことにした。「データの説明」ではなく「その数字の『奥』にある『経営の覚悟』を見える化する」ことについて。

## 1. 「成績の数字」から「そこに至る『判断の歴史』を逆算する」

投資家は「結果の数字」より「その結果に至るまでの『経営判断の積み重ね』」を見ています。AIに「売上3倍という結果の奥に『過去2年で何度の『判断の転換』があったか』『その転換のたびに『何を失って、何を得たか』を言語化してください」と頼みます。例えば「去年の『営業組織のフルリプレイス』という判断。一時的に売上が落ちたが『営業の質』が上がり、今の成長に繋がった」という「判断と責任」の物語が見えると「単なる成長」が「覚悟のある成長」に見えます。

## 2. 「失敗経験」を「学習の証」として提示する

投資家が最も信頼するのは「失敗からの『具体的な学習』」です。AIに「過去18ヶ月で『最も大きな失敗』を1つ挙げて、その失敗が『組織のどの部分を変えたか』『その変化が『現在の成長』にどう繋がっているか』を3段階で説明してください」と頼みます。「新商品開発に失敗した → 顧客開発プロセスを導入した → その結果『今の主力商品が生まれた』」という『失敗→学習→成果』の物語が見えると「この経営陣は『未来を作る』力がある」という信頼が生まれます。

## 3. 「数字の『意味』」を経営陣の『価値観』で翻訳する」

売上数字は「一般的な成長」に見える。でも「その成長は『顧客を大事にする経営』から出た」という『価値観の訳』が入ると「単なる成長」が「信頼できる成長」に見えます。AIに「売上3倍という結果を『投資家に見せるとき、その背景に『この経営陣が何を『最優先』にしているか』という『価値観』が見える説明』に変え直してください」と頼みます。例えば「売上3倍の裏に『顧客満足度も3倍』という数字がある。それは『短期利益より『顧客信頼』を優先する経営判断』の証だ」という『価値観の訳』が入ると「この会社は『急成長するが『倒れない』」という投資判断になります。

## 4. 「次の危機」を「経営陣がどう見ているか」を言語化する

投資家は「今の好調さ」より「次に起きる『不況』『競争激化』『技術変化』に『どう対応する覚悟があるか』」を見ています。AIに「来年、起きる可能性のある『業界の不況』『競争の激化』『新技術の登場』という3つのリスク。あなたの経営陣は『この3つに対して、どんな準備をしているか』『その準備が『どう会社を守るか』」を説明してください」と頼みます。危機への「備え」を見える化できる経営陣は「安心できる経営陣」に見えます。

## 体験談: 「数字の説明」から「覚悟の物語」へ

初回のIR説明会で、私は「売上3倍、利益黒字化」という数字だけを並べました。投資家の反応は「ふむふむ」で終わり。その後「実は経営判断の大転換があった」という『背景の物語』を後付けで説明したとき、投資家の目が変わりました。「あ、この経営陣は『短期数字より『会社の健全性』を優先しているんだ』という信頼」。その説明会の後、3社から投資提案が来ました。それ以来、IR資料は「数字」から「数字を支える『経営判断の物語』」に変えました。結果「数字は同じ」なのに「投資家の信頼度」が倍になりました。データを語るのではなく「経営陣の『何を信じているか』を語る」が、IRの本質だと分かりました。

## 🎯 上級プロンプト: 「失敗の『学習の物語』をIRに組み込む」法

投資家の信頼を引き出す時は、AIに次のように頼みます:
「過去の『最大の失敗』から『学んだこと』『その学習が『現在の経営』にどう活きているか』を『投資家が『この経営陣は『未来を作る力がある』と確信する』ような説明』に変えてください。」

このプロンプトで、IR説明は「数字の説明」から「経営陣の『信念を見せる』説明」に変わります。私はこれで「赤字だった時期の『顧客開発に100%の資源を投下した判断』が『今の黒字化に繋がった』という『覚悟の物語』を見える化し、それが『複数の投資家の『信頼』を生みました。
