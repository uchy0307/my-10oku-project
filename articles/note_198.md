# 【AI対話アプリ】200の問い #198「海外向けPR」

このアプリは「海外向けPR」のためのAI対話プロンプト集です。「日本文化」を「日本語のまま説明」しては、世界には伝わりません。「その文化が『世界のどんな価値観の隙間』を埋めるのか」を読む力で、日本の『小さな価値』を『世界の大きなニーズ』に翻訳するスキルを言語化します。

---


## 1. 「日本の価値」を「世界の『困りごと』」にマッピングする

日本文化のPRで最初にすべきは「日本人が思う『良さ』」を手放すこと。AIに「『江戸職人技』『手作りの温かさ』『世代継承』という『日本人が感じる価値』を、『今の世界が『何に困っているか』という視点から『言い直し』てください」と頼みます。すると「江戸職人技＝『深い技術の信頼』に翻訳され、それは『大量生産への違和感』への対抗軸」に見える。「世界の困りごと」に日本文化を宛がむことで「日本の『小さな価値』が『世界の『大きなニーズ』」に変わります。


## 2. 「文化説明」ではなく「感情体験」を約束する

「江戸職人技とは...」という歴史説明では、世界は読みません。AIに「『江戸の漆職人の茶碗』を『世界の忙しい20代女性』が『朝の一杯のお茶を飲むとき』に『何を感じるか』を『感覚的に』3段階で描写してください」と頼みます。その「感情の体験」が見えると「説明」ではなく「感動」に変わります。


## 3. 「『世界の言葉』で『本質』を説明し直す」

「ワビ」などの日本語は、世界には「謎の言葉」です。AIに「『わび・さび』という日本の美意識。これを『英語圏の人が『納得する言葉』で説明してください」と頼みます。「わび＝『imperfection in simplicity』(シンプルの中にある不完璧さの美しさ)」と訳されると「ああ、つまり『完璧さを求めるな』という哲学か」と世界が理解します。


## 4. 「『この文化を持つ人』をキャラクター化する」

抽象的な「文化」より「その文化を『生きている人』」を見ると世界は信頼します。AIに「『江戸漆職人の技を守る』人物を『世界の人が『そういう生き方もあるのか』と感動する』というプロフィール形式で書いてください」と頼みます。


## 体験談: 「『日本文化の説明』から『世界の心を掴む翻訳』へ」

最初のPRは「完璧な日本文化解説」でした。結果「アクセス数ほぼゼロ」。その後、AI指導を受けて「方針を180度変えた」。「江戸職人技」ではなく「『大量生産に疲れた世界が『求めている『一生ものの相棒』という視点」でPR。「翻訳」を入れた瞬間「SNSでバイラル」「世界中の工芸品マニアが注目」。同じ商品。違うのは「世界に『どう翻訳したか』」だけ。文化の「説明」から「感動への翻訳」で、PRの質が全く変わります。

海外向けPRで一番の落とし穴は『日本国内で通用する文脈』をそのまま英訳することです。AIに『この訴求文は、英語圏文化で誤解を生む可能性のある言い回しを指摘して』と頼むと、無自覚な前提が浮かび上がる。海外PRは翻訳ではなく、文化の橋渡しです。橋を渡る前に、AIに地図を描かせる癖をつけるのが安全です。

それでは、以下のステップで思考を深めていきましょう。まず、あなたが海外にPRしたい日本の対象（例: 伝統工芸品、特定の食品、アニメーション、温泉文化、ビジネス習慣など）を具体的に一つ挙げ、その基本的な内容を簡潔に説明してください。

あなたの回答を受け、次にAIは以下の質問を順に投げかけてください。私はその質問に答えることで、PR戦略を練っていきます。

4. これまでの対話を踏まえ、その日本の対象が世界のターゲット層の「大きなニーズ」にどのように応えるか、具体的なPRコンセプトやキャッチフレーズのアイデアをいくつか提案してください。

これらの質問へのあなたの回答が全て出揃ったら、AIはこれらの情報に基づき、海外向けPR戦略を具体的に実行するための次の一歩を3つ提案してください。提案は、ターゲット層へのアプローチ方法、メッセージングの調整、または具体的なプロモーション活動の方向性など、実践的な内容でお願いします。
