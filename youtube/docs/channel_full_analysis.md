## 7. ポジティブ要因 / ネガティブ要因 分析

### ★ バズ要因（再現可能パターン）

1. **「番外編」プレフィックス**
   人物単体より「現代人が思わずクリックする雑学」枠でブランド化。avg 33,982 = 偉人伝の 29 倍。
2. **タブー×身体×短尺の三位一体**
   切腹/斬首/避妊/生理/出産/お風呂 → 全て「江戸/侍時代の身体・性・死」。3〜5 分で完視聴を狙う。
3. **疑問形タイトル**
   「〜とは？」「〜の痛みとは？」「〜の意識」 → 検索意図と完全一致。
4. **タイトルに数値や形容詞を入れる**
   「四大人斬り」「5分でわかる」「総集編」 → 認知負荷を下げる。
5. **アルゴリズム的に「危ない題材ほど推奨される」現象を活用**
   切腹 92 万は年齢制限ありにも関わらず大バズ → むしろ年齢制限が品質シグナルとして機能している可能性。

### ✗ 不調要因

1. **「人物単発」の過剰投下**
   偉人伝 81 本投稿で総再生 9.5 万。1 本あたり 1,176 回。**「歴史好き視聴者の認知バイアス（同じ顔ぶれ ≠ 新規視聴）」** が効いている。
2. **20 分超の長尺**
   完視聴離脱 → アルゴリズム評価低下のスパイラル。坂本龍馬（69 分）すら 3,507 回。
3. **同一題材の細切れ**
   坂本龍馬を「少年/青年/立身/大成/最期」「完全版」「ロングバージョン」「龍馬の生涯」と 10 種類以上に分散。共食いで全体的に薄い。
4. **英語版（[Reading]）の機械翻訳**
   グローバル市場狙いだが英語コンテンツの SEO は別ゲーム。avg 62 回 → 全 7 本撤退検討。
5. **「動画固有の説明文」不在**
   Top 1 の説明欄は完全にチャンネル定型テンプレ。動画個別の検索フックを設定しないと「関連動画推奨」依存になる。
6. **新シリーズ「侍の美学」の動画長設計**
   28〜34 分は過去データ的に最も再生されないレンジ。

### 投稿頻度×再生数の関係
- 大半は「4 年前」「3 年前」（2022 春〜2023 春の 1 年）に集中投下 → その時期に集中力で投稿していた事が判明。
- 直近の **2 年間（2024〜2025）に新規動画ほぼ無し**、突如「侍の美学」5 本連投で復帰 → アルゴリズムは「休眠チャンネル」と認識している可能性。
- 教訓: **連続性のある投稿頻度** が再評価のレバー。

### C. サムネ設計の方向性

- 現状の **黄色和紙背景 + 肖像 + 赤太字キーワード** はチャンネルブランドとして機能している（リピーター認知に寄与）
- ただし **Top 3 のサムネ** は要再点検 → 「切腹の痛み」「斬首」「避妊」では肖像ではなく **抽象的アイコン（刀・絆創膏・本）が主**と推測される。バズ動画ほど **人物より概念をビジュアル化** している可能性。
- 改善案: 新作は **「太字キーワード × 衝撃的アイコン × 顔写真は2 番手」** に。
- **サムネ A/B テスト** を Youtube Studio で実装推奨（年齢制限ありの #1 でも 92 万 → サムネが強い証拠）。

### D. 配信時間帯の最適化

- 直近 5 本（侍の美学）は 12〜19 時間前 = 19 時〜深夜 1 時投稿 → 視聴時間帯としては平日夜なら可だが、競合（DOWNTOWN+ などのお笑い系）と被る
- 推奨: **金曜日 21:00 / 土曜日 18:00** に集中投稿（雑学ジャンルの視聴ピーク）
- 投稿頻度: **週 1 本** で 3 ヶ月継続し、アルゴリズム評価復帰を狙う
